【鑑別対象 20疾患】
■ 小球性貧血（MCV＜80）
 鉄欠乏性貧血（IDA）/ サラセミア / 鉄芽球性貧血 / 鉛中毒 / IRIDA

■ 正球性貧血（MCV 80-100）
 急性出血 / 慢性疾患貧血（ACD/AI）/ 腎性貧血 / AIHA / TMA
 機械的溶血 / 遺伝性溶血（HS・G6PD）/ 再生不良性貧血
 PNH / 骨髄浸潤 / PRCA

■ 大球性貧血（MCV＞100）
 ビタミンB12欠乏 / 葉酸欠乏 / アルコール・肝疾患関連
 甲状腺機能低下症 / MDS

--------------------------------------------------

【Red Flags】
- TTP/MAHA：血小板減少+溶血+破砕赤血球 → 治療を遅らせない
- 重度貧血+循環不安定 → Hb＜7で輸血検討
- 汎血球減少+芽球 → AA/MDS/白血病を疑い緊急紹介
- 中高年の新規IDA → 消化管悪性腫瘍を除外（上下部内視鏡）
- 神経症状+巨赤芽球性 → B12欠乏は不可逆あり、早期補充
