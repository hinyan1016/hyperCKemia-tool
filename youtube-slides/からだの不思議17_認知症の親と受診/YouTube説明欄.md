認知症の親を病院に連れて行くには｜受診を嫌がる場合の実践的アプローチ【からだの不思議 #17】

「もの忘れ外来に連れて行こうとしたら『私はボケていない』と拒否された」
「かかりつけ医に相談しようとしても、本人の前では何も言えない」
「何度試みてもうまくいかず、家族全員が疲弊してきている」

外来で非常によく聞く相談です。

この動画では、なぜ本人が受診を拒否するのかという医学的な背景と、
実際に外来でご家族にお伝えしている具体的な対処法を解説します。

━━━━━━━━━━━━━━━━━━━━━
🧠 受診を嫌がる3つの理由
━━━━━━━━━━━━━━━━━━━━━
① 病識低下（アノソグノシア）
　→ 脳の変化により「自分はおかしくない」と本当に感じている
　→ アルツハイマー型の 40〜81% にみられる

② スティグマへの恐れ
　→「施設に入れられる」「家族に迷惑をかける」
　→ 長年「一家の柱」だった世代ほど強い

③ 医療不信・先送り
　→「年をとれば誰でも忘れる」「今度でいい」

━━━━━━━━━━━━━━━━━━━━━
✅ 受診につなげる3つのアプローチ
━━━━━━━━━━━━━━━━━━━━━
1️⃣ 「認知症の検査」とは言わない
　→ 健康診断・脳ドック・血圧の薬のついで など別の目的を前面に

2️⃣ かかりつけ医を経由する
　→ 家族だけで先に相談 → 先生から本人に声かけをお願いする

3️⃣ まず家族だけで相談に行く
　→ もの忘れ外来・地域包括支援センターは患者不在でも相談できる

━━━━━━━━━━━━━━━━━━━━━
📋 もの忘れ外来の流れ
━━━━━━━━━━━━━━━━━━━━━
1. 問診（症状・日常生活への影響）
2. 神経心理検査（MMSE・MoCA、30分程度）
3. 血液検査（治療可能な原因の除外）
4. 頭部MRI（萎縮・脳血管病変の確認）
5. 結果と説明（1回で白黒つかないことも多い）

→ ベースラインの記録を残すこと自体に大きな意味がある

━━━━━━━━━━━━━━━━━━━━━
📞 相談窓口
━━━━━━━━━━━━━━━━━━━━━
・地域包括支援センター（無料・まず相談）
　「地域包括支援センター＋お住まいの市区町村名」で検索
・認知症疾患医療センター（専門拠点）
　かかりつけ医から紹介
・認知症の人と家族の会（家族支援）
　電話相談: 0120-294-456 / alzheimer.or.jp

━━━━━━━━━━━━━━━━━━━━━
📝 Take Home Message
━━━━━━━━━━━━━━━━━━━━━
1. 受診拒否の背景には「病識低下」という医学的原因がある。頑固なだけではない
2. 「認知症の検査」とは言わず、別の入口を使う
3. まず家族だけで相談できる（もの忘れ外来・地域包括支援センター）
4. 1回で白黒つかなくてよい。ベースラインの記録が将来につながる

━━━━━━━━━━━━━━━━━━━━━
🚨 こんな症状は救急へ（もの忘れ外来ではなく）
━━━━━━━━━━━━━━━━━━━━━
・急に（数日〜数週間で）認知機能が落ちた
・手足の麻痺・言語障害・激しい頭痛を伴う
・高熱・意識の変容がある
・強い幻視・幻聴で本人や家族が危険な状況
→ 脳卒中・脳炎・せん妄の可能性あり
