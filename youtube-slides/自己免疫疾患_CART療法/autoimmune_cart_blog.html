自己免疫疾患CAR-Tの全体像

神経内科医のための、できるだけ分かりやすい整理

自己免疫疾患に対するCAR-Tの本質は、「免疫を弱め続ける治療」から、「病気を作っているB細胞系を一度深く掃除して、免疫の設定を組み直す治療」への発想転換です。いま最も強い臨床証拠はSLE・強皮症・炎症性筋疾患など全身性自己免疫疾患で出ていますが、神経内科領域でもMG、NMOSD、MS、CIDP、stiff-person syndrome、自己抗体性脳炎などへ一気に広がっています。

1. まず結論

大づかみに言うと、自己免疫CAR-Tは次の3点で理解すると見通しが良くなります。

  • 第1に、標的は「抗体」そのものではなく、「抗体を作る系統の細胞」です。
  • 第2に、rituximabの延長線ではなく、もっと深い場所までB細胞系を消せる可能性がある治療です。
  • 第3に、神経内科で相性が良いのは、自己抗体やB細胞の関与が強く、しかも臓器障害がまだ可逆的な段階にある病型です。

現時点で神経内科で最も開発が進んでいるのはMGです。NMOSDは生物学的には非常に相性が良く、臨床試験も始まっています。MSは「中枢神経内のB細胞コンパートメントに本当に届くか」が核心で、初期のヒトデータはかなり面白いです。CIDPやGAD関連疾患、自己抗体性脳炎は、まだ症例報告レベルが中心ですが、すでに「効きうる疾患群」が見え始めています。[1][4]

2. なぜCAR-Tが自己免疫に効くのか

図1. CAR-T細胞療法の基本的な流れ
患者のT細胞を採取→遺伝子改変でCAR（キメラ抗原受容体）を導入→体外増殖→患者に再注入。がん領域では腫瘍細胞を標的とするが、自己免疫疾患では自己反応性B細胞を標的とする。 Image: Reyasingh56 via Wikimedia Commons / CC BY-SA 4.0

自己免疫疾患を単純化すると、B細胞が間違った設計図を持ち、形質芽細胞・形質細胞が自己抗体を作り続ける病気です。今までの多くの治療は、この工場の「出荷量」を減らす治療でした。ところがCAR-Tは、設計部門や工場そのものを壊しにいけます。だから、単なる一時的な鎮静ではなく、免疫の"誤学習"をリセットできるのではないかと期待されているわけです。[1]

この考え方を強く支持したのがSLEのデータです。2022年の報告では、難治性SLE 5例にCD19 CAR-Tを行い、全例が3か月でDORIS寛解に入り、しかもB細胞が戻った後も薬剤フリー寛解が持続したことが示されました[2]。戻ってきたB細胞は未熟・naive寄りで、以前の「病気を覚えたB細胞」がそのまま戻ったわけではありませんでした。これは、単なるB細胞減少ではなく、免疫の再起動が起きていることを示唆します。

その後、SLE・炎症性筋疾患・強皮症を含む15例のNEJM症例集積[3]、さらに24例のCASTLE試験[4]へと進み、多くの患者で免疫抑制薬やステロイドを止めても寛解・改善が続くことが示されました。神経内科医にとって重要なのは、「自己免疫をCAR-Tで長期寛解へ持ち込める」という原理が、もはや理論ではなくヒトで再現され始めた点です。

3. 標的の違いを1枚で整理する

図2. B細胞の活性化と分化
抗原刺激を受けたB細胞は、メモリーB細胞と形質細胞（抗体産生細胞）へ分化する。CAR-T療法では、この分化経路のどの段階を標的にするかで治療の性格が変わる。CD19は広範なB細胞系を、BCMAは形質芽細胞・形質細胞を主に標的とする。 Image: Bds0223 via Wikimedia Commons / CC BY-SA 4.0

B細胞系を「川上から川下へ」並べると、
naive B cell → memory B cell → plasmablast → plasma cell
となります。どこを狙うかで治療の性格が変わります。

標的 たとえ 主に狙う場所 何が得意か
CD20 現場の作業員 成熟B細胞 ふつうのB細胞を減らす
CD19 会社全体＋記憶倉庫 B細胞系を広く 免疫記憶ごと深くリセット
BCMA 抗体工場 plasmablast / plasma cell いま抗体を作る細胞を叩く
CD19+BCMA 倉庫＋工場の両方 上流と下流 より深い除去
CAAR-T 指名手配犯だけ狙う狙撃手 自己抗原特異的B細胞 正常免疫をなるべく残す
📋 ポイント
CD19は"広く深く"、BCMAは"抗体工場を直撃"、CAAR-Tは"犯人だけ"。神経内科で今もっとも現実味があるのはCD19、BCMA、CD19+BCMAで、CAAR-Tは次世代です。

4. rituximabと何が決定的に違うのか

ここが臨床的には一番重要です。

rituximabはCD20を標的にしますが、形質細胞は基本的にCD20陰性です。つまり、末梢血のB細胞がきれいに消えても、リンパ節や病変組織、骨髄などにいる「病気を支えるB細胞系」や「抗体工場」が残ることがあります。だから、rituximabでよく効く病気でも、"病気の根っこ"が残ることがあるわけです。

この違いを直接示したのが、自己免疫疾患での組織比較研究です[5]。末梢血では両方ともB細胞が見えなくても、CAR-T後はリンパ節でCD19陽性・CD20陽性B細胞が深く消え、非リンパ組織でもB細胞が大きく減る一方、rituximab後は組織内B細胞が残ることが示されました。

📋 重要な解釈
CAR-Tは「血液検査上のB細胞ゼロ」ではなく、組織の奥にいる病的B細胞まで掘り起こして消す可能性があります。CAR-Tがrituximabより"強い薬"というより、別の相の治療である理由です。

5. 神経内科の各論

5-1. MGは、いま最も面白い

図3. 神経筋接合部（NMJ）の構造
運動神経の軸索終末からアセチルコリン（ACh）が放出され、筋線維上のAChR（アセチルコリン受容体）に結合する。MGではこのAChRやMuSKに対する自己抗体が産生され、神経筋伝達が障害される。CAR-T療法はこれらの自己抗体を産生するB細胞・形質細胞を標的とする。 Image: Dake via Wikimedia Commons / CC BY-SA 3.0

MGでは、実は2つのCAR-T思想が並走しています。1つはDescartes-08のようなBCMA標的mRNA CAR-T、もう1つはCD19/BCMA二重標的のような古典的・深部型CAR-Tです。

Descartes-08の特徴は、外来投与、リンパ球除去前処置なし、mRNA型で一過性という点です。2023年のMG-001試験では、重篤なCRSや神経毒性はみられず、症状スケールの改善が最大9か月続きました[6]。12か月追跡でも、6週投与を受けた7例全例が3か月で改善し、5例は12か月時点でも反応を維持、再増悪した3例のうち2例は再投与で再び速やかに改善しました[7]。さらに、2026年には無作為化・二重盲検・プラセボ対照のphase 2b試験が公表され[8]、AURORAというphase 3試験も進行中です。

📋 臨床的な解釈
Descartes-08は、"1回で深く免疫をリセットする古典的CAR-T"というより、"抗体工場を外来で安全に間引く、再投与可能な細胞治療"に近い印象です。mRNA型で一過性、前処置不要、再投与で再反応が得られていることが、その見方を支えます。

一方、2025年のBCMA/CD19二重標的CAR-Tのphase 1試験では、難治性全身型MG 18例に単回投与が行われ、ICANSなし、CRSはgrade 1が39%のみでした[9]。180日でMG-ADLとQMGは大きく改善し、17例中14例がminimal manifestation、15例がステロイド中止、全例が非ステロイド系免疫抑制薬を中止し、AChR抗体陰性化も一部でみられました。こちらは、Descartes-08よりも「深いリセット」に近い絵です。

⚠️ MGの本質的な面白さ
同じCAR-Tでも"外来でやさしく工場を叩く型"と"深くリセットする型"が共存していることです。これは他の神経免疫疾患にも広がる可能性があります。

5-2. NMOSDは、生物学的にはかなり相性がよい

AQP4抗体陽性NMOSDは、自己抗体の病原性がかなり明確です。したがって、B細胞・形質細胞を深く叩く戦略は理屈が通りやすい疾患です。phase 1試験では、再発難治性AQP4-IgG陽性NMOSD 12例にBCMA CAR-Tが使われ、CRSはgrade 1–2のみ、中央値5.5か月追跡で11/12例が無再発、障害度とQOLも全体に改善し、AQP4抗体も低下傾向を示しました[10]。

もちろん、追跡期間はまだ短く、長期再発率や感染リスク、再投与戦略は未確立です。それでも、「自己抗体が病態の中心」「抗体除去だけでなく、抗体産生源を狙う意義が大きい」という意味で、NMOSDは自己免疫CAR-Tの適応候補としてかなり上位にいます。

5-3. MSは「中枢に届くか」が勝負

図4. オリゴデンドロサイトと髄鞘の構造
中枢神経系ではオリゴデンドロサイトが軸索に髄鞘（ミエリン）を形成する。MSではこの髄鞘が免疫攻撃により破壊（脱髄）される。進行型MSでは中枢神経内にB細胞が閉じこもった炎症が持続し、CAR-Tがこのコンパートメントに到達できるかが治療の鍵となる。 Image: Andrew c r via Wikimedia Commons / CC BY-SA 3.0

MS、とくに進行型MSでは、末梢だけでなく中枢神経内に閉じこもったB細胞炎症が問題です。ここにCAR-Tが届くなら、意味は非常に大きい。

実際、進行型MS 2例へのCD19 CAR-Tでは、CAR-Tが髄液内で検出されてもICANSは起きず、1例では髄液内抗体産生がday 64まで低下しました[11]。これは、「CAR-TがCNSコンパートメントに入り、少なくとも一部では機能している」ことを示す初期シグナルです。

さらに、progressive MSを対象にしたKYV-101試験や、CD19/BCMA二重標的AZD0120のMS試験も進んでいます。つまりMSでは、"本当にコンパートメント炎症を崩せるか"を臨床で検証する段階に入っています。

5-4. CIDPは「全員」ではなく「合うサブタイプ」がありそう

CIDPは異質性が大きい疾患群です。そのため、CAR-TがCIDP全体に効くというより、自己抗体・形質細胞依存性の強いサブタイプに効くと考えるのが自然です。

2025年の報告では、再発を繰り返す難治性CIDP 2例にBCMA CAR-Tが行われ、2例とも6か月以内に薬剤フリー寛解へ入り、1例は24か月以上寛解を維持、もう1例は重症COVID-19後に12か月で再発しました[12]。再発時には、病的B細胞の再活性化と自己抗体/ペプチドの再出現が示されました。

📋 示唆
CIDPは1つではない。B細胞・抗体依存性の強い患者では、CAR-Tが"根治的に近い寛解"を狙えるかもしれない。逆に、軸索障害が前景で、自己抗体の証拠が弱い例にはそのまま当てはめにくいでしょう。

5-5. stiff-person syndrome、GAD関連疾患、自己抗体性脳炎

この領域はまだ症例報告が中心ですが、内容はかなり刺激的です。

stiff-person syndromeの重症難治例ではCD19 CAR-T後に歩行速度が100%以上改善し、歩行距離が50 m未満から6 km超へ増加、ベンゾジアゼピンも40%減量され、毒性は低グレードCRSにとどまりました[13]。

DAGLA抗体関連脳炎の難治例では、単回CD19 CAR-T後に臨床スコアが改善し、血清・髄液抗体が低下し、髄液OCBが陽性から陰性化しました[14]。GAD65抗体関連小脳失調症でも、2026年の初報でday 90に血清GAD65抗体が95%低下し、day 30から失調が改善、毒性はgrade 1 CRSのみでした[15]。

⚠️ 注意
ここはまだ"有望"であって"確立"ではありません。症例報告は強い仮説を生みますが、適応選択・長期予後・再投与・感染管理はこれからです。

6. 神経内科医が実際に考えるべき4つの問い

問い1：その病気は、本当にB細胞/自己抗体が主役か

CAR-Tは万能の免疫抑制ではありません。B細胞が病態の中心であるほど、相性が良い。NMOSD、AChR抗体陽性MG、自己抗体性脳炎の一部はこの条件を満たしやすい。進行型MSやCIDPは、病型・サブタイプの見極めが重要です。

問い2：障害はまだ可逆的か

自己抗体を止めても、壊れた組織が戻るとは限りません。CAR-Tは炎症や病的免疫のドライバーを止める治療であって、神経再生そのものではありません。したがって、固定化した不可逆障害が前景の時期より、まだ機能改善余地のある時期のほうが理屈上は向いています。

問い3：CD19か、BCMAか、両方か

ざっくり言えば、

  • CD19は「広く深くリセットしたい」とき
  • BCMAは「抗体工場を叩きたい」とき
  • CD19+BCMAは「供給源も工場も両方叩きたい」とき

という発想です。

さらに将来のCAAR-Tは、MuSK-MGやNMDAR脳炎のように自己抗原が明確な病気で、犯人B細胞だけを狙う戦略です。MuSK-CAAR-Tは前臨床で抗MuSK B細胞を選択的に減らし、総B細胞や総IgGは保ちました[16]。NMDAR-CAAR-Tも前臨床で抗NMDAR B細胞を選択的に除去しました[17]。

問い4：患者はCAR-Tの物流と毒性に耐えられるか

古典的な自家CAR-Tは、通常、アフェレーシス、製造待機、前処置のリンパ球除去化学療法、CRS/ICANS監視が必要です。つまり、良い薬というだけでなく、"良いセンター"が必要な治療です。Descartes-08が特別なのは、ここを大きく簡略化した点です。

7. 自己免疫CAR-T特有の副作用にも注意する

がん領域で有名な毒性はCRSとICANSですが、自己免疫領域では別の現象も見えています。

39例の自己免疫患者を解析した研究では、既存病変臓器に一致した局所反応が77%に起こり、これをLICATS（Local Immune Cell-Associated Toxicity Syndrome）と名付けています[18]。多くはgrade 1–2で、中央値10日で出現し、11日ほどで自然軽快しました。著者らは、病変臓器から免疫細胞が掃除される過程に伴う反応と解釈しています。

⚠️ 神経内科への示唆
神経内科でこれがどう見えるかは、まだ直接データが乏しいです。ただ、もし神経根、筋、髄膜、脳辺縁系など既存病変部で似た現象が起これば、再燃との区別が難しいはずです。今後の診療ではかなり重要になると考えられます。

8. 次に来るもの

未来の方向は大きく2つです。

Allogeneic / off-the-shelf化

健康ドナー由来のCD19 CAR-Tを使った研究では、重症筋炎・強皮症の3例で2週以内に完全B細胞枯渇、6か月で深い寛解、CRSや重篤有害事象なしが報告され[19]、SLEでも3例でGvHD・CRS・ICANSなしに寛解が示されました[20]。これが進めば、待機時間と物流の壁が大きく下がります。

Precision CAR-T（CAAR-T）

MuSK-MGやNMDAR脳炎のように標的自己抗原が明確な病気では、将来は正常B細胞を温存しながら、犯人クローンだけを消す方向へ進む可能性があります[16][17]。これは、神経免疫にとって特に相性のよい進化です。

9. まとめ

自己免疫CAR-Tを一言で言うと、「免疫抑制」から「免疫リセット」への転換です。

神経内科の視点では、次のように覚えると整理しやすいです。

  • MGは最前線で、しかも「やさしいBCMA mRNA型」と「深いCD19/BCMA型」が並走している。
  • NMOSDは生物学的に非常に相性がよい。
  • MSはCNSコンパートメント攻略が核心。
  • CIDPやGAD/自己抗体性脳炎は、合う患者を見つける時代に入りつつある。

そして最も大事なのは、CAR-Tはrituximabの強化版ではなく、病的B細胞系の深部除去という別の治療概念だということです。
