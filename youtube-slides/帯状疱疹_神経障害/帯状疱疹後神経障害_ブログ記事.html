帯状疱疹＝皮疹と痛み、で終わっていませんか？　実は運動麻痺や排尿障害まで起こしうる"神経の病気"です。

帯状疱疹は水痘・帯状疱疹ウイルス（VZV）が神経節で再活性化することで発症します。多くの医師は皮疹と痛みを主症状として認識していますが、障害は感覚神経だけでなく運動神経にも及ぶことがあります。本記事では、臨床で遭遇する帯状疱疹の神経障害を整理します。

帯状疱疹の発疹はデルマトームに沿って分布する（Wellcome Collection, CC BY 4.0）

1. 神経障害の全体像

帯状疱疹の本態は神経節炎（ganglionitis）です。炎症は以下の部位へ波及します。

障害部位 主な症状
後根神経節 皮疹・神経痛
感覚神経 感覚障害
前角・運動神経 運動麻痺
脳神経 顔面神経麻痺など
自律神経 排尿障害、腸管障害
臨床で特に重要な3つの障害

1. 帯状疱疹後神経痛（PHN）
2. 帯状疱疹関連運動麻痺
3. 脳神経障害

2. 感覚神経障害 ― 最も頻度が高い

急性期の神経痛

皮疹と同時または先行して出現する痛みで、以下の特徴があります。

  • 電撃痛
  • Burning pain
  • アロディニア

原因は後根神経節炎・神経線維障害・中枢感作です。

帯状疱疹後神経痛（PHN）

皮疹治癒後3か月以上続く痛みをPHNと定義します。

年齢 PHN発症率
50歳未満 約5%
60歳以上 10〜20%
80歳以上 30%近く

帯状疱疹の進行 — 休眠ウイルスが神経線維を伝って皮膚に到達する（Public Domain, FDA）

PHNの危険因子

高齢 / 激しい急性期痛 / 皮疹範囲が広い / 免疫抑制

予後：

  • 1年以内改善 … 約50%
  • 数年持続 … 約20%
  • 永続 … 少数

3. 運動神経障害（Zoster paresis）― 見逃されやすい

帯状疱疹の3〜5%に運動麻痺が起こります。前角または運動神経根への炎症波及が原因で、多くは皮疹から2〜3週間後に出現します。

デルマトーム（皮膚分節）の分布図 — 麻痺の領域は罹患神経に対応する（Public Domain, Mikael Häggström）

主な臨床型

1. 四肢の分節性麻痺（最多）

頚神経 → 上肢麻痺
胸神経 → 腹筋麻痺（腹壁膨隆＝pseudohernia）
腰神経 → 下肢麻痺
2. 顔面神経麻痺（ラムゼイ・ハント症候群）

顔面神経麻痺 ＋ 外耳道疱疹 ＋ 難聴 ＋ めまい
3. 横隔神経麻痺（稀）

呼吸困難、横隔膜挙上
4. 膀胱直腸障害

仙骨神経の感染 → 尿閉・排尿困難・便秘

4. 運動麻痺の予後 ― 比較的良好

予後 割合
完全回復 約50〜70%
部分回復 約20〜30%
後遺症 約10%
回復までの期間

軽症例：3か月
一般的：6〜12か月
重症例：2年以上

予後不良因子：高齢 / 免疫抑制 / 広範囲皮疹 / 強い神経障害

5. 検査

運動麻痺が疑われる場合に有用な検査：

検査 所見
MRI 神経根の造影効果・神経腫脹
神経伝導検査 CMAP低下、軸索障害
髄液 軽度の細胞数増加・蛋白増加

6. 治療

抗ウイルス薬（急性期）

  • アシクロビル / バラシクロビル / ファムシクロビル
  • 72時間以内の開始が理想

神経痛治療

  • プレガバリン、ガバペンチン
  • 三環系抗うつ薬
  • 神経ブロック

運動麻痺

運動麻痺の治療方針

基本は保存療法 ＋ リハビリ
顔面神経麻痺ではステロイド併用

まとめ ― 覚えておくべき5つのポイント

# ポイント
1 帯状疱疹は「皮膚の病気」ではなく神経の病気
2 運動麻痺は3〜5%で起こる
3 麻痺は皮疹後2〜3週間で出現する
4 予後は比較的良い（多くは回復）
5 PHNは高齢ほど発症率が高い

帯状疱疹では感覚神経障害・帯状疱疹後神経痛・運動麻痺など多様な神経障害が起こります。特に運動麻痺は見逃されやすいですが、四肢麻痺・腹壁麻痺・顔面神経麻痺・排尿障害などを呈します。しかし多くの症例では数か月〜1年で回復するとされています。
