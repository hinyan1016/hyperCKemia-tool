# 帯状疱疹は「皮膚の病気」ではない ― 知っておくべき神経障害の全体像

> 帯状疱疹＝皮疹と痛み、で終わっていませんか？　実は**運動麻痺や排尿障害**まで起こしうる"神経の病気"です。

---

## はじめに

帯状疱疹は**水痘・帯状疱疹ウイルス（VZV）**が神経節で再活性化することで発症します。多くの医師は皮疹と痛みを主症状として認識していますが、障害は**感覚神経だけでなく運動神経にも及ぶ**ことがあります。

本記事では、臨床で遭遇する帯状疱疹の神経障害を整理します。

## 1. 神経障害の全体像

帯状疱疹の本態は**神経節炎（ganglionitis）**です。炎症は以下の部位へ波及します。

| 障害部位 | 主な症状 |
|:---|:---|
| 後根神経節 | 皮疹・神経痛 |
| 感覚神経 | 感覚障害 |
| 前角・運動神経 | **運動麻痺** |
| 脳神経 | 顔面神経麻痺など |
| 自律神経 | 排尿障害、腸管障害 |

臨床で特に重要なのは次の3つです。

1. **帯状疱疹後神経痛（PHN）**
2. **帯状疱疹関連運動麻痺**
3. **脳神経障害**

---

## 2. 感覚神経障害 ― 最も頻度が高い

### 急性期の神経痛

皮疹と同時または先行して出現する痛みで、以下の特徴があります。

- 電撃痛
- Burning pain
- アロディニア

原因は後根神経節炎・神経線維障害・中枢感作です。

### 帯状疱疹後神経痛（PHN）

皮疹治癒後**3か月以上続く痛み**をPHNと定義します。

| 年齢 | PHN発症率 |
|:---|:---|
| 50歳未満 | 約5% |
| 60歳以上 | 10〜20% |
| 80歳以上 | **30%近く** |

![帯状疱疹後神経痛のメカニズム](https://www.mayoclinic.org/-/media/kcms/gbs/patient-consumer/images/2017/08/15/20/41/sn7_shinglescau)
*▲ 脊髄と罹患神経の関係（Mayo Clinic）*

**危険因子：** 高齢、激しい急性期痛、皮疹範囲が広い、免疫抑制

**予後：**
- 1年以内改善 … 約50%
- 数年持続 … 約20%
- 永続 … 少数

---

## 3. 運動神経障害（Zoster paresis）― 見逃されやすい

帯状疱疹の**3〜5%**に運動麻痺が起こります。前角または運動神経根への炎症波及が原因で、多くは**皮疹から2〜3週間後**に出現します。

![デルマトーム（皮膚分節）の分布図](https://media.springernature.com/lw685/springer-static/image/chp%3A10.1007%2F978-1-4939-2462-2_18/Me)
*▲ デルマトーム（皮膚分節）の分布図 — 麻痺の領域は罹患神経に対応する*

### 主な臨床型

**① 四肢の分節性麻痺（最多）**
- 頚神経 → 上肢麻痺
- 胸神経 → 腹筋麻痺（腹壁膨隆＝pseudohernia）
- 腰神経 → 下肢麻痺

**② 顔面神経麻痺（ラムゼイ・ハント症候群）**
- 顔面神経麻痺＋外耳道疱疹＋難聴＋めまい

**③ 横隔神経麻痺**（稀）
- 呼吸困難、横隔膜挙上

**④ 膀胱直腸障害**
- 仙骨神経の感染 → 尿閉・排尿困難・便秘

---

## 4. 運動麻痺の予後 ― 比較的良好

| 予後 | 割合 |
|:---|:---|
| 完全回復 | **約50〜70%** |
| 部分回復 | 約20〜30% |
| 後遺症 | 約10% |

**回復までの期間：**
- 軽症例：3か月
- 一般的：6〜12か月
- 重症例：2年以上

**予後不良因子：** 高齢、免疫抑制、広範囲皮疹、強い神経障害

---

## 5. 検査

運動麻痺が疑われる場合に有用な検査：

- **MRI** — 神経根の造影効果・神経腫脹
- **神経伝導検査** — CMAP低下、軸索障害
- **髄液** — 軽度の細胞数増加・蛋白増加

---

## 6. 治療

### 抗ウイルス薬（急性期）
- アシクロビル / バラシクロビル / ファムシクロビル
- **72時間以内の開始が理想**

### 神経痛治療
- プレガバリン、ガバペンチン
- 三環系抗うつ薬
- 神経ブロック

### 運動麻痺
- 基本は**保存療法＋リハビリ**
- 顔面神経麻痺では**ステロイド併用**

## まとめ ― 臨床で覚えておくべき5つのポイント

| # | ポイント |
|:---|:---|
| 1 | 帯状疱疹は**「皮膚の病気」ではなく神経の病気** |
| 2 | 運動麻痺は**3〜5%**で起こる |
| 3 | 麻痺は**皮疹後2〜3週間**で出現する |
| 4 | 予後は比較的良い（**多くは回復**） |
| 5 | PHNは**高齢ほど発症率が高い** |

帯状疱疹では感覚神経障害・帯状疱疹後神経痛・運動麻痺など**多様な神経障害**が起こります。特に運動麻痺は見逃されやすいですが、四肢麻痺・腹壁麻痺・顔面神経麻痺・排尿障害などを呈します。しかし多くの症例では**数か月〜1年で回復する**とされています。
