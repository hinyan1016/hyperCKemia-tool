# 「その症状、大丈夫？」シリーズ — デザインドキュメント

## 概要

脳神経内科医が監修する**患者・一般向け**医学教育シリーズ第2弾。
日常で感じる「気になる症状」の正体・危険度・受診の目安を、脳神経内科医の視点でわかりやすく解説する。

> **重要**: 本シリーズは既存の医師向け鑑別診断ツール（medical-ddx-tools/）とは**完全に独立したシリーズ**である。医師用ツールへのリンクは一切設置しない。

## コンセプト

**「不安解消型」** — 読者/視聴者の「この症状、放っておいて大丈夫？」という不安に応える

### ターゲット
- 30-60代の健康意識層（患者本人）
- 症状をGoogle検索する層（「〇〇 原因」「〇〇 何科」）
- 家族の症状が気になる層（認知症・運転等）
- **医療従事者はターゲット外** — 専門用語を避け、平易な日本語で記述

### 差別化ポイント
- 大手医療メディア（メディカルノート等）は病名ベース → 本シリーズは**症状ベース**
- 脳神経内科医の一般向けコンテンツは**ブルーオーシャン**（精神科・整形外科に比べ希少）
- ブログ → **患者向けセルフチェックツール**連携という独自ファネル
- GoogleのE-E-A-Tアップデートにより、現役医師の一次情報は検索上位に有利

### 医師用ツール（medical-ddx-tools）との明確な違い
| | 医師用ツール（既存） | 患者向けツール（本シリーズ） |
|---|---|---|
| 対象 | 医療従事者 | 患者・一般市民 |
| 目的 | 鑑別診断の支援 | 「受診すべきか？」「何科？」の判断支援 |
| 用語 | 医学専門用語 | 平易な日本語 |
| 出力 | 疾患リスト・確率 | 緊急度（すぐ受診/早めに受診/様子見）+ 推奨診療科 |
| 設置場所 | medical-ddx-tools/ | symptom-checker/（新規・独立） |
| 相互リンク | なし | なし |

## 各回の共通構成

### ブログ記事の構成テンプレート
1. **導入** — 共感を呼ぶ症状描写（「こんな経験ありませんか？」）
2. **症状の正体** — 医学的メカニズムをわかりやすく解説
3. **よくある原因** — 頻度の高い原因から順に
4. **危険なサイン** — 「こんな場合はすぐ受診」の明確な基準
5. **受診の目安と診療科** — 何科に、いつ行くべきか
6. **セルフケア・予防** — 家庭でできる対処法
7. **まとめ** — ポイント3-5個の箇条書き
8. **参考文献** — DOI付きリンク

### YouTube動画の構成テンプレート
- **スライド**: pptxgenjsで作成（BIZ UDPGothic / Segoe UI）
- **台本**: スライドと1対1対応、1スライドあたり200-300字目安
- **概要欄**: タイムスタンプ + 関連リンク + ブログ記事リンク

### SEO戦略
- タイトル形式: 「{症状の日常表現}」― {不安への回答} を脳神経内科医が解説
- シリーズ統一タグ: `その症状大丈夫`
- 各記事のmeta description: 検索クエリを含む120字以内

## 制作ワークフロー（各回共通）

1. **SEO競合分析** — 検索KWの競合記事を確認、差別化ポイントを特定
2. **見出し構成** — ブログ記事のH2/H3構成を決定
3. **文献調査** — PubMed/ガイドラインから根拠となる文献を収集
4. **ブログ記事作成** — HTML、はてなブログ投稿規約準拠
5. **スライド作成** — pptxgenjs、視覚検証（LibreOffice→PDF→PNG）
6. **台本作成** — スライドと1対1対応
7. **概要欄作成** — タイムスタンプ + 関連リンク
8. **はてなブログ投稿** — Playwright MCP経由

## 患者向けセルフチェックツール新規作成予定

### ツールの設計方針
- **対象**: 患者・一般市民（医療従事者ではない）
- **言語**: 専門用語を使わず、平易な日本語で質問

- **UI**: やさしい配色、大きめのボタン、スマホファースト
