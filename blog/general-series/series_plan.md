# 一般向けシリーズ「脳神経内科医が答える からだの不思議」

## コンセプト

一般の人が日常で感じる「からだの不思議」を、脳神経内科医の視点で科学的に解説する。
「正座の立ちくらみ」「金縛り」記事の路線を拡大し、SNS拡散・検索流入の両方を狙う。

## 戦略の根拠

- 直球の疾患名キーワード（「パーキンソン病 症状」等）はクリニックHP・製薬会社・公的機関が独占。個人ブログは上位10件に皆無
- 日常の「なぜ？」系・判断系キーワードは競合が少なく、SNSでの拡散も期待できる
- 一般向け入口 → 既存の専門記事・診断ツール → YouTube への導線を構築する

## 記事フォーマット

### 構成テンプレート

```
■ タイトル（30字以内、疑問形 or 数字入り）
■ リード文（3行以内、共感→問題提起→「この記事でわかること」）
■ 目次（自動生成）
■ 本文（H2×3〜5セクション、各セクション300-500字）
  - H2: 「こんな経験ありませんか？」（共感パート）
  - H2: 「脳の中で何が起きているか」（メカニズム解説）
  - H2: 「こんなときは要注意」（受診基準）
  - H2: 「まとめ — 脳神経内科医からのアドバイス」
■ 参考文献（DOIリンク付き）
■ 関連記事リンク（既存の専門記事・診断ツールへ）
```

### 仕様

- **文字数**: 2,000〜3,000字（専門記事より短く、読み切りやすい長さ）
- **文体**:
  - 専門用語は使ってOKだが、必ず（）で平易な説明を添える
  - 「〜という報告があります」「〜が示唆されています」で断定を避ける
  - 「実際の診療でよく聞かれるのですが」「外来で多い質問です」で臨床経験（E-E-A-Tの"Experience"）を見せる
- **JS記法**: var / function(){} / 文字列連結（既存ブログ規約に準拠）

## SEO・拡散施策

1. **著者プロフィールの強化** — 「脳神経内科専門医」「所属学会」をブログのAboutページに明記（E-E-A-T最重要）
2. **記事公開 → X(Twitter)で要点3行 + リンク投稿** — はてなブックマーク初動を稼ぐ
3. **FAQ構造化データ** — 「Q. しびれは何科？」「A. 〜」形式でGoogle検索のリッチスニペット表示を狙う
4. **内部リンク戦略** — 一般向け記事 → 既存の専門記事・診断ツールへ誘導リンクを必ず設置
5. **シリーズまとめ記事** — 10本溜まったら「脳神経内科医が答える からだの不思議 まとめ」を作成
6. **YouTube連動** — 反応の良かった記事をスライド動画化（既存パイプラインで対応可能）
