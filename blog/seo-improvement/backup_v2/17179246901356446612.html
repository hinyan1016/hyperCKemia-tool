エグゼクティブサマリー

薬剤抵抗性てんかん（DRE）に対する植込み型ニューロモジュレーション（VNS・DBS・RNS）は、発作を「ゼロ」にする治療というより、発作頻度・重症度を下げて生活を立て直す緩和的（palliative）治療として位置づけられます。

🔵 VNS（迷走神経刺激）

左頸部迷走神経→脳幹（孤束核）を入口に、ノルアドレナリン/セロトニンなどの広域修飾系を介して脳全体の興奮性・同期性を下げる。最も「広く・浅く」効かせる戦略。

🟢 DBS（脳深部刺激／ANT-DBS）

視床前核（ANT）を標的に、辺縁系・視床皮質ネットワーク（Papez回路）を調整する。SANTE試験で短期効果が示され、長期で効果が増すパターン（例：7年で中央値75%減）が一貫して報告。

🟠 RNS（反応性神経刺激）

発作焦点（通常1–2か所）に電極を置き、頭蓋内脳波（ECoG）から「発作に向かうパターン」を検出して刺激を返すclosed-loop（閉ループ）戦略。9年で中央値75%減・レスポンダー率73%まで改善。

📋 日本における承認・保険状況
  • VNS：2010年より保険適用。
  • ANT-DBS：2023年に適応追加・保険適用（月の詳細は資料間で差異あり）。
  • RNS：少なくとも2025年時点で国内未承認（公的情報として確認）。

理解のための前提：DREと主要概念

DRE（薬剤抵抗性てんかん）は、国際抗てんかん連盟（ILAE）の合意定義として「適切に選択され十分量・十分期間使われた2剤（単剤/併用）で、持続的な発作消失が得られない」が広く用いられます[1]。一方、日本てんかん学会の外科適応を意識した基準としては「2〜3種類の適切な薬でも2年以上抑制されない」という考え方も示されています（"いつ専門施設に紹介するか"という臨床上の目的が少し異なります）[2]。

用語 この記事での意味 実データ上の代表例
発作減少率 ベースライン（術前/導入前）と比べた発作頻度の減少（%） RNSの9年追跡で中央値75%減
レスポンダー率 発作が50%以上減った人の割合 ANT-DBSは5年で68%
「効果は時間とともに増す」 導入直後より数年後の方が成績が良い傾向 VNS・RNS・ANT-DBS、いずれも長期で改善を確認

VNS・DBS・RNSを一言で分けると、open-loop（一定の刺激を続ける）かclosed-loop（発作の兆候を検出して反応する）か、そして「刺激が届く範囲」が違います[3]。

作用機序の全体像

「電気刺激で発作が減る」には、原理的に少なくとも3つの層があります。まず、解剖学的標的（どこを刺激するか：末梢神経・視床・発作焦点）。次に、回路・神経伝達物質レベル（どのネットワーク/修飾系を動かすか）。最後に、短期 vs 長期の神経生理変化（急性期は同期性破綻・抑制、慢性期はネットワーク再編＝可塑性で徐々に改善）という「時間の軸」です[4]。

VNS（迷走神経刺激）の機序

解剖学的標的は左頸部迷走神経で、植込み型ジェネレータから間欠刺激を入れます[5]。VNSの特徴は、迷走神経の求心性線維が脳幹の孤束核（NTS）に入り、そこから上行性に広く影響する点です。回路・神経伝達物質レベルでは、NTSからノルアドレナリン系・セロトニン系・アセチルコリン系などを介して、大脳皮質に「広汎な安定化」がもたらされます[5]。

短期の生理変化としては、動物実験で皮質神経細胞の過分極と自発放電低下、さらに発作誘発状態での同期性低下（過剰同期をほどく）が示されています[5]。臨床側から見ると、VNSの作用は「局所をピンポイントに遮断する」より「ネットワークの興奮性・同期性を下げる」方向で理解するとわかりやすいです。

長期の変化は、効果が治療継続で増大し、およそ2年程度で伸びてその後安定するという日本の実施ガイドラインの記載が有名です[6]。この「長期で効きが増す」性質はAANのガイドライン更新でも示唆されており、臨床的には「設定調整＋時間」を治療の一部として組み込む発想になります[7]。

VNS 回路イメージ（概念図）

VNSの神経回路は以下の流れで理解できます：

左頸部迷走神経への間欠刺激 → 孤束核（NTS） → 青斑核（ノルアドレナリン系）・縫線核（セロトニン系）・傍腕核など → 視床・皮質ネットワーク → 大脳皮質の興奮性/同期性↓ → 発作頻度・重症度↓（緩和的効果）

この流れは「NTS→広域修飾系→皮質の安定化」という説明に沿っており[5]、VNSが焦点不明瞭な症例にも広く使用される根拠となっています。

DBS（脳深部刺激）の機序：なぜ「視床前核（ANT）」なのか

現在、てんかんDBSで最もエビデンスが厚い標的は視床前核（ANT）で、SANTE試験で有効性が示されました[8]。NICEも、成人の難治てんかんに対するDBSのエビデンスは刺激部位で異なる（ANTは限定的だが一定の根拠、ANT以外は研究枠）と整理しています[9]。

ANTを理解する鍵は、ANTが辺縁系の環状回路（Papez回路）に位置することです[10]。SANTEの長期成績論文でも、機序は完全には不明としつつ「ネットワーク調整」として議論されています[11]。

📋 ANT-DBSの臨床的作用原理 発作活動が「視床—皮質」「辺縁系—皮質」連絡を介して拡がるなら、その中継点（ANT）に一定パターンの刺激を入れて、病的同期・伝播を起こしにくくするというのが基本的な考え方です[10]。短期変化：SANTEの盲検期で「刺激群の方が減る」ことが示されています[8]。長期変化：5年で中央値69%減、7年で75%減まで改善しうるという「時間とともに増す」パターンが一貫しています[11]。
Papez回路とANTの位置関係（概念図）

Papez回路内でのANTの役割：

海馬 → 脳弓 → 乳頭体 → 視床前核（ANT） ← DBS調整 → 帯状回 → 嗅内皮質 → 海馬

ANTは辺縁系ネットワークの中継点として位置し、DBS刺激により辺縁系ネットワークの興奮性を調整→発作の伝播/同期↓→発作頻度↓という経路が想定されます[10]。

RNS（反応性神経刺激）の機序：closed-loopの強み

RNSは、発作焦点に留置した電極から頭蓋内脳波（ECoG）を読み続け、パターン一致時に刺激するclosed-loopです[12]。NeuroPaceのシステムは、成人の部分（焦点）発作で、焦点が1–2か所のケースを対象に、ランダム化比較試験で有効性が示されています[13]。

短期の原理は「発作に向かう異常活動を検出→すぐ刺激→発作を小さくする/止める」です[14]。一方で長期の原理は、9年追跡で効果が増していく事実（中央値75%減、レスポンダー率73%）から、反復介入によるネットワーク再編（可塑性）を含むと考えるのが自然です[15]。「検出して刺激する」「切除不要で可逆的」という特徴が、RNS機序の重要な整理です[16]。

RNS 閉ループ（closed-loop）動作フロー

焦点（1–2か所）に電極留置 → 頭蓋内脳波を常時計測 → 発作に先行するパターンを検出？

  • 検出なし → 計測継続
  • 検出あり → 検出に反応して刺激 → 急性効果（異常放電/同期を崩す）→ 発作の停止/軽減

慢性効果：反復介入によってネットワーク/発作閾値が変化 → 長期の発作負荷↓

VNS：大規模メタ解析（Englotら）では、平均で約45%の発作減少（3–12か月で約36%、1年以上で約51%）が報告されています[23]。DBS（視床ターゲット）：2025年メタ解析では、全体の平均発作減少は約62%、ANTで約64%、CMNで約69%とされ、レスポンダー率も60%台が中心です（研究の質・対象の異質性に注意）[24]。RNS：2022年メタ解析では、平均発作減少率68%、平均レスポンダー率68%、合併症率18.9%とまとめられています（出版バイアスの指摘あり）[25]。

どう選ぶか：臨床的選択アルゴリズム

選択の主軸は「発作焦点の数と切除可能性」「侵襲性」「必要な作用範囲」です（"どれが一番効くか"だけでは決まりません）[29]。まずは外科的根治（切除/離断/焼灼）の可能性を精査し、難しい場合に緩和的な刺激療法を検討するという順序は日本の資料でも繰り返し強調されています[30]。

📋 VNSが向きやすい状況 焦点が多い/局在が不明瞭/全般発作を含むなどで「脳内に電極を入れて局所を狙う」より「まず広域に効かせたい」ケース[31]。日本では保険適用で、実施には資格・体制が必要です[32]。
📋 ANT-DBSが向きやすい状況 主に成人の焦点てんかんで切除が難しい場合。特に側頭葉てんかんなど辺縁系が強く関与するタイプが臨床解説で言及されます（ただし個別評価が必須）[33]。
⚠️ RNSが向きやすい状況（海外の標準／国内は未承認） 焦点が1–2か所に絞れているが、機能野（eloquent area）などで切除が難しい、あるいは両側内側側頭葉などで切除がためらわれるケースに適合しやすいとされています[34]。ただし日本では国内未承認のため、現時点では保険診療での実施はできません[35]。

まとめと今後の展望

ひとことでまとめ

🔵 VNS

「脳幹—広域修飾系経由で、脳の"興奮しやすさ"を下げる。侵襲が低く、じわじわ効く。」

🟢 ANT-DBS

「発作ネットワークの中継点（視床前核）を調整し、年単位で発作を減らす。」

🟠 RNS

「焦点（1–2か所）を見張って、発作の兆候に反応して刺激する。長期で成績が伸びる。」

今後の展望（臨床に関係するところ）

3モダリティとも「長期で効きが増す」傾向があり、今後の焦点は誰がレスポンダーになりやすいか（予測因子・バイオマーカー）と、最適な刺激条件/標的の個別化です[38]。DBSはANT以外（CMなど）も研究が進み、ガイドラインでも「標的によって根拠が違う」と整理されています[9]。日本ではRNSは未承認で、導入に向けた要望・議論や、今後の企業治験参加の言及が公的/大学資料で確認できます[35]。
