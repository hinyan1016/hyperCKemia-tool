0. 出発点 — 並列で除外する6疾患

🚨 急性胸痛で並列除外すべき6疾患[1][2]
① 急性冠症候群（ACS）
② 肺塞栓症（PE）
③ 急性大動脈症候群（AAS）
④ 急性心膜炎 / 心嚢液貯留
⑤ 気胸
⑥ 肺炎

とくに鋭い胸膜性胸痛では、PEが最も頻度の高い重篤原因で、救急受診例の5〜21%にみられます。一方で、刺すような痛みや呼吸で悪化する痛みはACSらしくはないものの、症状の性質だけでACSは否定できません。[1][2][13]

1. 最初の10分

📋 初期評価チェックリスト[1][3][4]
・バイタル、SpO₂、意識、ショック/呼吸不全の有無を即確認
・12誘導ECG：来院10分以内に取得
・高感度トロポニン：初回だけでなく再検を前提に置く
・胸部X線：PE以外の原因（肺炎、気胸など）の拾い上げに早期から並行
・どの分岐に見えても、ACSの評価はECGとトロポニンで並走させる

2. まず除外する6疾患

疾患 まず疑う手がかり 初期の次の一手
ACS 圧迫感、放散痛、労作関連、冷汗、嘔気、呼吸困難。ただし鋭い痛みでも除外不可 ECG + 高感度トロポニン再検。痛みの質だけで切らない
PE 胸膜性痛、呼吸困難、頻脈、低酸素、喀血、片脚腫脹/疼痛、VTE危険因子 PERC/Wellsで前確率評価 → D-dimer / CTPA
AAS 突然・最大時発症の激痛、胸背部痛、失神、血圧/脈拍左右差、神経脱落症候 ADD-RSで整理 → 高疑いなら大動脈造影CT
急性心膜炎 鋭い胸痛、吸気/臥位で悪化、前屈で軽快、心膜摩擦音、びまん性ST上昇/PR低下 4項目中2項目以上で診断。心エコー考慮
気胸 突然の片側胸膜性痛、呼吸困難、片側呼吸音低下 胸部X線または超音波/CT。不安定なら緊張性気胸を最優先
肺炎 発熱、咳、喀痰、ラ音、胸膜性痛 画像上の新規浸潤影を確認。症状だけでは確定しない

3. 分岐アルゴリズム

Step 1：不安定かどうかを先に決める

🚨 不安定の徴候[1][3][4]
ショック、重度低酸素、持続する虚血性ECG変化、失神、著明な神経脱落、呼吸循環不安定
→ 蘇生しながら ACS / 高リスクPE / AAS / 緊張性気胸 / 心タンポナーデを同時進行で考える

Step 2：安定例の分岐

手がかり 優先する分岐
突然・最大時発症の胸背部痛、血圧左右差、脈拍差、神経脱落、失神、結合組織病/大動脈病変 → AAS分岐
胸膜性痛 + 呼吸困難、低酸素、頻脈、喀血、片脚症状、VTE危険因子 → PE分岐
前屈で軽快、臥位で悪化、摩擦音、びまん性ST上昇/PR低下 → 急性心膜炎
突然の片側胸膜性痛 + 呼吸困難 + 片側呼吸音低下 → 気胸
発熱・咳・喀痰・ラ音 → 肺炎
⚠️ 重要な原則[1][13]
上のどれに見えても、ACS分岐は最後まで外さない。鋭い痛み、圧痛、呼吸で変動する痛みでも、症状だけではACSは否定できない。

4. すぐ使う判断ルール

PE分岐

PERCは「全体としてPE低疑いで、他の診断も十分あり得る」場合だけ使うルールです。[6]

📋 PERCの8項目（全項目陰性なら追加のPE検査を省けることがある）[6]
① 年齢 <50 ② 脈拍 <100 ③ SpO₂ ≥95% ④ 片脚腫脹なし
⑤ 喀血なし ⑥ エストロゲン使用なし ⑦ 4週以内の手術/外傷入院なし ⑧ 既往VTEなし

Wells 2-level PE score

項目 点数
DVTを示唆する所見 3
PE以外の診断が考えにくい 3
心拍数 >100/分 1.5
3日超の不動化 or 4週以内の手術 1.5
DVT/PE既往 1.5
喀血 1
悪性腫瘍 1
📋 Wells scoreの解釈[4][5]
・>4点：PE likely → CTPA（造影不適ならV/Q）へ
・≤4点：PE unlikely → D-dimer → 陽性なら画像、陰性ならPEは後退

AAS分岐 — ADD-RS（Aortic Dissection Detection Risk Score）

領域 代表例
高リスク素因 Marfan/結合組織病、家族歴、既知の大動脈弁疾患、最近の大動脈操作、既知の胸部大動脈瘤
高リスク疼痛 突然発症、強い痛み、ripping/tearing
高リスク診察所見 脈拍/血圧左右差、灌流障害、神経脱落症候、新規大動脈弁逆流雑音、低血圧/ショック
⚠️ D-dimerはAASの単独除外検査ではない[7][8][9]
・ADvISED研究：ADD-RS ≤1 かつ D-dimer <500 ng/mL → failure rate 0.3%[9]
・しかし高疑い例ではD-dimerに頼らず大動脈造影CTを優先する

急性心膜炎の診断基準

📋 以下の4項目中2項目以上で急性心膜炎と診断[10]
① 典型的な胸膜性胸痛（前屈で軽快、臥位で悪化）
② 心膜摩擦音
③ ECG変化（びまん性ST上昇 and/or PR低下）
④ 新規または増悪する心嚢液貯留

5. 見逃しやすい点

落とし穴 正しい認識
「胸膜性」「刺すような痛み」＝ 良性 まずPEを考え、同時にACSも切らない[1][2]
神経症状が前景の胸背部痛 胸痛/背部痛 + 血圧差があればAASを強く考える[7]
急性心膜炎で胸部X線が正常 胸部X線正常でもあり得る → ECGと心エコーが重要[10]
発熱 + 胸膜性痛 ＝ 肺炎 画像上の浸潤影が鍵。症状だけでは確定しない[11]
痛みの質でACSを否定する 鋭い痛み、圧痛、呼吸で変動する痛みでもACSは否定できない[1][13]
