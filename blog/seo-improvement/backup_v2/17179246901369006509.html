はじめに：甲状腺機能検査の落とし穴

甲状腺機能異常は日常診療で遭遇頻度が高く、プライマリ・ケア外来の約5%で甲状腺機能検査に異常が見つかるとされます[1]。スクリーニングではTSH単独測定が広く行われますが、これには重大な落とし穴があります。

TSH単独スクリーニングの盲点
中枢性甲状腺機能低下症（下垂体性・視床下部性）では、TSHが低値〜正常にとどまるため、TSH単独では見逃される。FT4低値にもかかわらずTSHが上昇しないパターンを常に念頭に置く[2]。
・下垂体腫瘍、シーハン症候群、リンパ球性下垂体炎、頭蓋内照射後など
・有病率は甲状腺機能低下症全体の約1/1,000と稀だが、副腎不全合併時に甲状腺ホルモン補充を先行すると副腎クリーゼを誘発するリスクがある[3]

したがって、臨床的に甲状腺機能低下を疑う場合や、下垂体疾患のリスクがある患者では、TSHに加えてFT4を同時測定することが推奨されます[2]。

TSH × FT4/FT3 パターン分類（7パターン）

甲状腺機能検査の異常は、TSHとFT4/FT3の組み合わせから以下の7パターンに分類できます。パターンを認識することが鑑別診断の第一歩です[1][2]。

パターン TSH FT4 FT3 主な鑑別
1. 顕性甲状腺中毒症 低下 高値 高値 バセドウ病、機能性結節、甲状腺炎
2. T3優位型中毒症 低下 正常 高値 バセドウ病初期、機能性結節、T3製剤内服
3. 潜在性甲状腺中毒症 低下 正常 正常 軽症バセドウ、機能性結節、LT4過量
4. 顕性甲状腺機能低下症 高値 低値 低値 橋本病、術後/RAI後、薬剤性
5. 潜在性甲状腺機能低下症 高値 正常 正常 橋本病初期、ヨウ素過剰、回復期
6. 中枢性甲状腺機能低下症 低〜正常 低値 低値 下垂体腫瘍、シーハン症候群、ICI関連
7. TSH産生下垂体腺腫（TSHoma） 正常〜高値 高値 高値 TSH産生腺腫、甲状腺ホルモン不応症
実践的ポイント
・パターン7（FT4高値なのにTSH抑制されない）は不適切TSH分泌症候群（SITSH）と呼ばれ、TSH産生腺腫と甲状腺ホルモン不応症の鑑別が重要[4]
・NTI（non-thyroidal illness / euthyroid sick syndrome）では、急性期にTSH低下→FT3低下→FT4低下と変動し、回復期にTSH一過性上昇を認める。重症患者では検査解釈に注意[1]
・薬剤（ビオチン、ヘパリン、フェニトインなど）による検査干渉も鑑別に含める

甲状腺中毒症の鑑別（破壊性 vs 産生過剰）

TSH低下＋FT4/FT3高値を確認したら、次のステップは破壊性甲状腺中毒症（甲状腺組織の炎症・破壊による一過性ホルモン放出）と産生過剰型（甲状腺の自律的ホルモン合成亢進）の鑑別です。この区別は治療方針を決定する上で極めて重要です[5]。

鑑別の鍵：甲状腺シンチグラフィ（99mTc / 123I）
・取り込み亢進（びまん性）→ バセドウ病
・取り込み亢進（限局性 hot nodule）→ 機能性結節/TMNG
・取り込み低下→ 破壊性甲状腺炎（亜急性/無痛性）、薬剤性
シンチが施行できない場合はTRAb＋甲状腺エコー＋臨床所見で判断する[5][6]

バセドウ病（Graves' disease）

甲状腺中毒症の最多原因（60〜80%）[5]。TSH受容体に対する自己抗体（TRAb）が甲状腺を持続的に刺激し、ホルモン産生過剰を引き起こします。

項目 所見
自己抗体 TRAb陽性（感度95%以上）[5]
甲状腺 びまん性腫大、エコーで血流増加
眼症状 眼球突出、複視、眼瞼後退（Graves' ophthalmopathy）
シンチ びまん性高取り込み
治療 抗甲状腺薬（MMI/PTU）、RAI、手術

機能性甲状腺結節 / 多結節性甲状腺腫（TMNG）

結節が自律的にホルモンを産生します。高齢者に多く、ヨウ素欠乏地域では甲状腺中毒症の主要原因となります[6]。

・TRAb陰性
・シンチでhot nodule（周囲は抑制）
・治療：RAIまたは手術（抗甲状腺薬は根本治療にならない）

亜急性甲状腺炎（subacute thyroiditis / de Quervain thyroiditis）

ウイルス感染が先行し、甲状腺の炎症・破壊により一過性の中毒症を呈します[7]。

特徴 所見
症状 有痛性甲状腺腫大、発熱、頸部放散痛
炎症マーカー CRP/ESR著明高値（ESR > 50 mm/h が多い）
シンチ 取り込み低下〜消失
経過 中毒期（1〜2か月）→ 一過性低下期 → 回復（大半は正常化）
治療 NSAIDs、重症例はステロイド（抗甲状腺薬は無効）

無痛性甲状腺炎 / 産後甲状腺炎

自己免疫性の破壊性甲状腺炎です。圧痛を伴わない点で亜急性甲状腺炎と区別されます[7]。

・CRP/ESR正常〜軽度上昇（炎症が軽い）
・TPOAb陽性が多い
・シンチ：取り込み低下
・産後甲状腺炎は産後1年以内に発症し、約20〜30%が永続性低下症に移行[8]
・中毒期はβ遮断薬で対症療法（抗甲状腺薬は無効）

薬剤性甲状腺中毒症

鑑別上特に重要な薬剤を以下にまとめます。

薬剤 機序 鑑別ポイント
アミオダロン（AIT） Type 1：ヨウ素誘発性（産生過剰）
Type 2：破壊性甲状腺炎		 Type 1：基礎疾患あり、エコーで血流増加
Type 2：正常甲状腺、血流低下
混合型も多い[9]
免疫チェックポイント阻害薬（ICI） 破壊性甲状腺炎（中毒期→低下症） 抗PD-1/PD-L1で頻度高い。中毒期は一過性で低下症に移行することが多い[10]
ヨウ素（造影剤など） Jod-Basedow現象（ヨウ素誘発性中毒症） 基礎に結節性甲状腺腫がある患者で起こりやすい[9]
アミオダロン甲状腺障害（AIT）の鑑別
・AIT Type 1：既存の甲状腺疾患（結節/バセドウ）に過剰ヨウ素が加わり産生過剰。エコーで血流増加。抗甲状腺薬＋過塩素酸カリウムで治療
・AIT Type 2：直接的な甲状腺細胞障害による破壊性甲状腺炎。エコーで血流低下。ステロイドで治療
・実臨床では混合型が多く、判別困難な場合はステロイド＋抗甲状腺薬の併用も考慮[9]

甲状腺機能低下症の鑑別

橋本病（慢性甲状腺炎 / Hashimoto's thyroiditis）

甲状腺機能低下症の最多原因（ヨウ素充足地域）[1]。自己免疫による甲状腺の慢性炎症・線維化が進行します。

・TPOAb陽性（感度90〜95%）、TgAb陽性
・エコー：びまん性低エコー、不均一
・年間約2〜4%が顕性低下症に進行（TPOAb陽性＋TSH高値の場合）[11]
・LT4補充療法が標準治療

医原性低下症

・甲状腺全摘/亜全摘後：術後は全例でLT4補充が必要
・RAI（放射性ヨウ素）治療後：バセドウ病治療後、数か月〜数年で低下症に至ることが多い
・外照射後：頭頸部放射線治療後に発症しうる

薬剤性甲状腺機能低下症

薬剤 機序 注意点
リチウム（Li） ヨウ素取り込み・ホルモン放出阻害 投与患者の約20〜30%で低下症[12]
アミオダロン ヨウ素過剰による Wolff-Chaikoff 効果 ヨウ素充足地域ではAIHが最多。LT4補充で対応可[9]
ICI 自己免疫性甲状腺炎の惹起 抗PD-1で5〜10%。破壊性中毒期を経ずに低下症で発見されることもある[10]
TKI（チロシンキナーゼ阻害薬） 甲状腺への直接障害 スニチニブで頻度高い[12]

中枢性甲状腺機能低下症

見逃し注意：FT4低値 + TSH低〜正常
・下垂体腫瘍、シーハン症候群、リンパ球性下垂体炎、ICI関連下垂体炎、術後・放射線後
・TSHは基準範囲内でも「不適切に低い」ことが診断の手がかり
・下垂体MRI + 他の下垂体前葉ホルモン評価（ACTH、GH、LH/FSH、PRL）が必須[2][3]
・副腎不全の合併を除外してからLT4を開始すること（副腎クリーゼ予防）

潜在性甲状腺機能異常（subclinical thyroid dysfunction）

FT4/FT3が正常範囲内でTSHのみ異常を示す状態です。治療適応の判断が臨床上しばしば問題になります。

潜在性甲状腺機能低下症

状況 治療方針
TSH > 10 mIU/L LT4補充を推奨（顕性低下症への進行率が高い）[11][13]
TSH 4.5〜10 + 症状あり LT4補充を考慮（3〜6か月のトライアル）[13]
TSH 4.5〜10 + TPOAb陽性 進行リスク高い → 定期フォロー or 補充考慮[11]
TSH 4.5〜10 + 妊娠中/妊娠希望 LT4補充を推奨（ATA 2017ガイドライン）[8]
70歳以上 + TSH 4.5〜10 経過観察が優先（TRUST試験で症状改善なし）[14]

潜在性甲状腺中毒症

・TSH < 0.1 mIU/L（Grade II）：心房細動・骨折リスク増加のエビデンスがあり、治療を推奨（特に65歳以上、心疾患、骨粗鬆症リスク）[15]
・TSH 0.1〜0.4 mIU/L（Grade I）：リスク因子がなければ6〜12か月ごとの経過観察
・まず2〜3か月後に再検し、持続性を確認してから治療を決定[15]

Red Flags：甲状腺クリーゼと粘液水腫性昏睡

甲状腺クリーゼ（thyroid storm）

致死的緊急症 ─ Burch-Wartofskyスコアで評価
甲状腺クリーゼは未治療/不十分治療の甲状腺中毒症に感染・手術・外傷などのストレスが加わり発症します。致死率10〜30%とされ、臨床診断に基づく即座の治療開始が必要です[16]。
Burch-Wartofskyスコア（主要項目）
体温 37.2〜37.7℃：5点 / 37.8〜38.2℃：10点 / ... / ≥40℃：30点
中枢神経 軽度興奮：10点 / せん妄・精神症状：20点 / 痙攣・昏睡：30点
消化器 下痢：10点 / 嘔吐：10点 / 黄疸：20点
心血管 頻脈（HR 100〜109：5点 ... ≥140：25点）/ 心房細動：10点 / 心不全（軽：5点 / 重：15点）
誘因 あり：10点
判定 ≥45点：クリーゼ強く疑う / 25〜44点：切迫クリーゼ / <25点：クリーゼ否定的[16]
甲状腺クリーゼの治療（同時並行で開始）
1. 抗甲状腺薬：PTU大量投与（末梢でのT4→T3変換も阻害）
2. 無機ヨウ素：PTU投与1時間後に開始（Wolff-Chaikoff効果でホルモン放出抑制）
3. β遮断薬：プロプラノロール（T4→T3変換阻害作用もあり）
4. ステロイド：ヒドロコルチゾン（相対的副腎不全に対応＋T4→T3変換阻害）
5. 全身管理：冷却、輸液、誘因の治療[5][16]

粘液水腫性昏睡（myxedema coma）

甲状腺機能低下症の最重症形態 ─ 致死率25〜60%
高齢者、冬季に好発。意識障害、低体温、徐脈、低換気、低Na血症を呈します[17]。

・誘因：感染、寒冷曝露、鎮静薬、LT4中断、手術
・治療：
　- LT4静注（loading dose 200〜400 μg → 維持 50〜100 μg/日）
　- ヒドロコルチゾン（副腎不全合併の除外前に開始）
　- 保温（積極的な外部加温は末梢血管拡張→ショックのリスク）
　- 全身管理（換気補助、低Na補正、誘因治療）[17]

粘液水腫性昏睡の低Na血症
甲状腺機能低下によるADH分泌亢進（SIADH様）＋自由水排泄障害が原因。甲状腺ホルモン補充で改善するが、重症低Naでは高張食塩水を検討[17]。
