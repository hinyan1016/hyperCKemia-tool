# ポリニューロパチーの鑑別診断と必要な検査：総合ガイド ポリニューロパチーは多くの疾患で起こる複合的な末梢神経障害です。本記事では原因から検査まで、診断アプローチを詳しく解説します。適切な鑑別診断と早期介入が治療成績向上の鍵となります。

1. はじめに

ポリニューロパチーは複数の末梢神経が同時に障害される疾患群です。手足のしびれや痛み、筋力低下などの症状で発症し、多くの患者さんが神経内科や整形外科を受診します。原因は多岐にわたり、糖尿病やアルコール性のような頻度の高いものから、早期治療が生命予後を左右する希少疾患まであります[1]。本記事では、ポリニューロパチーの主な原因と鑑別のポイント、そして必要な検査について解説します。

ポリニューロパチーの鑑別において最も重要なのは、「可逆性」「進行性」「緊急治療が必要か」という3つの視点で優先順位を考えることです。このアプローチによって、治療可能な原因を見逃さず、適切な介入が可能になります。

2. ポリニューロパチーとは

ポリニューロパチーは、複数の末梢神経が障害される状態を指します。典型的には、最も長い神経から障害される「長さ依存性」の形態を取り、両側対称性に足先から症状が始まり、次第に上行していきます[1]。主な症状には以下があります：

  • 感覚症状：しびれ、痛み、異常感覚、感覚低下
  • 運動症状：筋力低下、筋萎縮
  • 自律神経症状：起立性低血圧、発汗異常、排尿障害、便秘など

3. ポリニューロパチーの主な原因

ポリニューロパチーには様々な原因があります。以下にカテゴリー別に主な原因を詳しく解説します。

3.1 代謝・内分泌性

糖尿病性ニューロパチー

日本国内で最も頻度が高い原因です[2]。長期間の高血糖によって神経が障害されます。主に感覚神経が侵され、「手袋・靴下型」の感覚障害が特徴的です。痛みやしびれが夜間に強くなる傾向があります。

末期腎不全（尿毒症）

腎機能が著しく低下した際に尿毒素が蓄積し、神経障害を引き起こします。透析患者さんに多く見られます。

甲状腺機能異常

甲状腺機能低下症では神経伝導速度の遅延、甲状腺機能亢進症では主に筋力低下が見られることがあります。

3.2 中毒性

アルコール性ニューロパチー

慢性的な大量飲酒により発症します。アルコールの直接毒性とビタミンB1欠乏の両方が関与します。痛みを伴う感覚障害が特徴的です。

薬剤性ニューロパチー

抗がん剤（プラチナ製剤、タキサン、ビンカアルカロイドなど）、抗結核薬（イソニアジド、エタンブトール）、抗レトロウイルス薬などで発症することがあります。投与中止により改善することもありますが、永続的な障害を残すこともあります。

環境曝露

有機溶剤や重金属（鉛、ヒ素、水銀）への職業曝露によるニューロパチーもあります。詳細な職業歴の聴取が重要です。

3.3 栄養欠乏・代謝異常

ビタミン欠乏

ビタミンB1、B6、B12の欠乏は重要な原因です[4]。アルコール依存症、胃切除後、長期のPPI（プロトンポンプ阻害薬）使用などで発症リスクが高まります。特に、ビタミンB12欠乏では亜急性連合性脊髄変性症を併発することがあります。

ビタミンB12欠乏は、胃切除後やPPI長期使用患者、厳格な菜食主義者などでリスクが高まります。MCV（平均赤血球容積）上昇を伴う貧血が見られる場合は、積極的に疑いましょう。早期補充治療により神経障害の進行を防止できます。

葉酸欠乏

葉酸欠乏もニューロパチーの原因となることがあります。貧血を伴うことが多いです。

銅欠乏

亜鉛の過剰摂取により相対的な銅欠乏が生じ、ニューロパチーを発症することがあります。

3.4 免疫介在性／炎症性

慢性炎症性脱髄性多発ニューロパチー（CIDP）

自己免疫性の脱髄性ニューロパチーで、徐々に進行する筋力低下が特徴です[3]。免疫療法に反応するため、早期診断が重要です。

CIDPは免疫療法に反応する治療可能なニューロパチーです。2ヶ月以上かけて進行する対称性の筋力低下、深部腱反射の低下、神経伝導検査での脱髄所見などが特徴的です。早期診断と治療介入が機能予後を大きく改善します。

ギラン・バレー症候群（GBS）亜型

急性に発症する自己免疫性ニューロパチーです。先行感染後数週間で発症することが多く、一般的には単相性の経過をたどります。

血管炎関連ニューロパチー

ANCA関連血管炎やHBV関連血管炎などで多発ニューロパチーを呈することがあります。多発単神経炎の形態をとることが多いです。

POEMS症候群

形質細胞の増殖に伴う希少な傍腫瘍症候群で、多発ニューロパチーに加え、臓器腫大、内分泌異常、M蛋白、皮膚変化などを伴います。VEGF（血管内皮増殖因子）が上昇していることが特徴です。

3.5 感染性

HIV関連ニューロパチー

HIVそのものによる神経障害と抗レトロウイルス薬による中毒性ニューロパチーが存在します。

HBV/HCV関連

肝炎ウイルス感染に伴う免疫複合体による血管炎性ニューロパチーを発症することがあります。

その他の感染症

ライム病、ハンセン病（らい病）、ジフテリア、ジカウイルス感染症、SARS-CoV-2感染症などでもニューロパチーを発症することがあります。渡航歴や皮疹の有無などが診断の手がかりになります。

3.6 遺伝性

シャルコー・マリー・トゥース病（CMT）

最も一般的な遺伝性ニューロパチーで、若年発症で緩徐に進行します[4]。特徴的な足部変形（槌趾、高足弓）がしばしば見られます。

家族性アミロイドポリニューロパチー（ATTRv）

トランスサイレチン遺伝子変異によるアミロイドーシスで、自律神経障害を伴うことが多く、心筋症も合併します。早期診断と治療が予後を左右します。

ミトコンドリア病

MELASなどのミトコンドリア病でもニューロパチーを呈することがあります。筋症状や中枢神経症状、代謝性アシドーシスなどを伴うことが多いです。

3.7 浸潤・沈着性

アミロイドーシス

AL型（免疫グロブリン軽鎖由来）やATTRwt型（野生型トランスサイレチン由来）のアミロイドーシスは、進行性の感覚・自律神経障害を特徴とします。心肥大や体位性低血圧を伴うことが多いです。

進行の速い感覚障害と自律神経障害の組み合わせが見られる場合は、アミロイドーシスを強く疑いましょう。特に心不全症状や不整脈、体位性低血圧を伴う場合は要注意です。早期診断と治療介入が生命予後を大きく改善します。

サルコイドーシス

全身性のサルコイドーシスの一症状としてニューロパチーを呈することがあります。

悪性リンパ腫・白血病

腫瘍細胞の神経への直接浸潤によりニューロパチーを発症することがあります。

3.8 腫瘍関連・傍腫瘍

パラネオプラスティックニューロパチー

抗Hu抗体や抗CV2抗体などの自己抗体が関与する傍腫瘍性ニューロパチーがあります。原発腫瘍の発見前にニューロパチーが先行することもあります。

腫瘍浸潤

腫瘍の直接浸潤によるニューロパチーも存在します。

3.9 その他全身疾患

結合組織病

SLE（全身性エリテマトーデス）、RA（関節リウマチ）、シェーグレン症候群などでニューロパチーを発症することがあります。

クリオグロブリン血症

クリオグロブリン血症は、低温で沈殿するタンパク質が血管を閉塞し、神経障害を引き起こします。HCV感染症との関連が知られています。

高クリスタリン血症

高クリスタリン血症による微小血管閉塞もニューロパチーの原因となります。

3.10 特発性

詳細な病歴聴取と検査を行っても、20～30％のケースでは原因が特定できません[1]。これらは特発性ポリニューロパチーと分類されます。

4. 実臨床で見落とさないためのポイント

ポリニューロパチーの原因は非常に多岐にわたりますが、実臨床で重要なのは治療可能な原因を見逃さないことです。以下に重要なポイントをまとめます。

鑑別診断の基本方針：

  1. まず「治療可能」「進行抑制可能」な原因（ビタミン欠乏、薬剤性、CIDP、血管炎、アミロイドーシス）を早期に除外する
  2. 既往歴・家族歴・薬剤歴・飲酒歴・職業曝露・旅行歴を聴取で網羅的にチェックする
  3. 神経伝導検査（脱髄 vs 軸索）と自律神経検査で絞り込み、必要に応じて神経生検や遺伝子検査を実施する

全例で少なくとも以下の採血を推奨します：

  • 血糖／HbA1c、腎機能、甲状腺機能
  • ビタミンB1・B12、銅
  • 血清電気泳動（M蛋白）
  • ANCA、補体、HIV/HBV/HCV 抗体

以下の症状・状況では早急な対応が必要です：

  • 進行が速い・無痛感覚障害＋自律神経障害ではアミロイドーシスや POEMS を念頭に早期専門施設紹介
  • 糖尿病・アルコール依存症など頻度の高い背景にも二次性ニューロパチーが隠れている可能性があるため、"common disease bias" に注意
  • 特発性として経過観察する場合も、半年～1年毎に再評価し新規症候が出ないか確認

5. ポリニューロパチーの検査・診断アプローチ

ポリニューロパチーが疑われる場合、以下のステップで検査を進めていきます。

1. 詳細な病歴聴取と身体診察

症状の発現様式（急性、亜急性、慢性）、分布パターン（対称性、非対称性）、感覚・運動・自律神経症状の有無、併存疾患、家族歴などを詳細に聴取します。神経学的診察では、筋力、深部腱反射、感覚検査（触覚、痛覚、温度覚、振動覚、位置覚）、自律神経機能評価などを行います。

2. 血液検査

全てのポリニューロパチー患者で以下の基本的な血液検査を行います：

  • 一般血液検査：血算、肝機能、腎機能、電解質
  • 血糖値、HbA1c
  • 甲状腺機能検査（TSH、FT4）
  • ビタミンB1、B6、B12、葉酸
  • 血清免疫固定電気泳動（M蛋白検索）
  • 自己抗体検査：ANCA、抗核抗体、抗SS-A/B抗体、リウマトイド因子
  • 感染症スクリーニング：HIV、HBV、HCV

臨床的に疑われる場合には、追加で以下の検査も考慮します：

  • 銅、セルロプラスミン
  • VEGF（POEMS症候群疑い時）
  • クリオグロブリン
  • 傍腫瘍性抗体（抗Hu抗体、抗CV2抗体など）
  • 重金属スクリーニング（職業曝露歴がある場合）

3. 神経生理学的検査

神経伝導検査（NCS）と針筋電図検査（EMG）は、ポリニューロパチーの診断において非常に重要です[3]。これらの検査により、以下の分類が可能になります：

  • 軸索型 vs 脱髄型
  • 感覚優位型 vs 運動優位型 vs 混合型
  • 大径線維優位型 vs 小径線維優位型

4. その他の補助検査

  • 自律神経機能検査：心電図RR間隔変動、交感神経皮膚反応、チルト試験など
  • 皮膚生検：小径線維ニューロパチーの診断に有用
  • 神経生検：血管炎、アミロイドーシス、CIDP、遺伝性ニューロパチーなどの診断に有用
  • 遺伝子検査：CMTなどの遺伝性ニューロパチーが疑われる場合
  • 全身検索：胸腹部CT、PET-CTなど（傍腫瘍性、アミロイドーシス、サルコイドーシスなどが疑われる場合）

6. 主な治療アプローチ

ポリニューロパチーの治療は原因に応じて異なります。

原因治療

  • 代謝性・内分泌性：血糖コントロール、透析、甲状腺機能の正常化
  • 中毒性：原因物質の回避（アルコール、薬剤、環境毒素）
  • 栄養欠乏：不足しているビタミンや微量元素の補充
  • 免疫介在性：ステロイド、免疫グロブリン静注療法、血漿交換、免疫抑制剤
  • 感染性：抗菌薬、抗ウイルス薬
  • 遺伝性：症状緩和療法、ATTRvではトランスサイレチン安定化薬やRNAi治療
  • 浸潤・沈着性：原疾患の治療（化学療法、臓器移植など）
  • 腫瘍関連：原発腫瘍の治療、免疫療法

対症療法

痛みを伴うニューロパチーに対しては以下の薬剤が使用されます[5]：

  • プレガバリン、ガバペンチン（カルシウムチャネルα2δリガンド）
  • デュロキセチン、アミトリプチリン（セロトニン・ノルアドレナリン再取り込み阻害薬）
  • カルバマゼピン、オキシカルバゼピン（ナトリウムチャネル遮断薬）
  • オピオイド（重度の痛みの場合）

生活指導のポイント：

  • 感覚障害がある場合は足の観察を毎日行い、傷や潰瘍の早期発見に努める
  • 転倒予防のための環境整備と適切な歩行補助具の使用
  • アルコール摂取の制限または中止
  • 適切な栄養摂取と必要に応じたサプリメントの使用
  • 規則的な運動によるADL維持と筋力トレーニング

7. まとめ

ポリニューロパチーは「頻度が高いが見逃せない可逆的原因」と「希少だが治療介入で予後が変わる原因」の両面があります。鑑別の要は系統的な聴取とスクリーニング検査、そして"treatable neuropathy"を先に押さえることです。原因不明の場合でも定期的な再評価が重要であり、専門医への相談も検討すべきです。
