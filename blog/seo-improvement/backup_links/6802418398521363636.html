日本語の音声認識技術は急速に進化し、パソコンでもスマートフォンでも自然に話した言葉をテキスト化できる時代になりました。ここでは、WindowsとmacOSに搭載されている標準機能と、精度や利便性に優れたサードパーティ製ツールの特徴を比較し、目的別におすすめを紹介します。医療現場では記録の迅速化やハンズフリー操作が重要であり、音声入力ツールの選択が作業効率に直結します。

1. パソコンで使える音声入力

1.1 Windows 11/10 に標準搭載された「音声入力」

Windows 11/10 にはオンライン音声認識を使った音声入力機能が標準搭載されています。日本語キーボードを選んでWindowsロゴキー＋Hを押すか、タッチキーボードのマイクアイコンをタップすると音声入力UIが起動し、マイクが青色の状態で話せばテキスト化されます。

⚠️ 注意
Microsoftのクラウド音声認識を利用するためインターネット接続が必須であり、オフラインでは使えません。句読点は「句読点」や「改行」などの日本語コマンドを話すことで入力できます。

無料で使えるものの、オンライン依存や専門用語の誤認識があるため、長文の医学記録や報告書には精度の高いツールが望ましいでしょう。

1.2 macOS の「音声入力」(旧ディクテーション)

macOSでもシステム設定の「キーボード」で音声入力を有効にし、挿入ポイントを指定した状態でFn(⌨)キーを押すかメニューの「編集→音声入力を開始」を選ぶと、マイクのアイコンが現れます。Apple Silicon搭載機では音声入力中にキーボード操作も併用でき、句読点や絵文字も音声で指定できます。

自動句読点機能があり、30秒間話さないと自動的に終了しますが、時間制限はないため長文の入力にも利用できます。ただしこちらも基本的にオンライン認識であり、難解な医学用語は誤認識しやすい点に注意が必要です。

1.3 高精度なサードパーティ製ツール

ツール 対応OS オンライン/オフライン 費用 主な特徴
Aqua Voice Windows/Mac (モバイル専用アプリは無し) オンライン（独自サーバー利用） 有料（サブスク） 独自の融合法認識とクライアント側のコンテキストエンジンにより遅延約450msの高速変換を実現し、メールやコードなど用途に応じて自動フォーマットを行う。複数のモデルを融合して発話の意図を理解し、自然言語コマンドに応じて文章の修正や引用の追加などを自動で行う。SiriやGoogleボイス入力より17倍少ない誤り率とされ、日本語を含む49言語に対応。現在はMac/Windows向けに提供されており、スマートフォン専用アプリは存在しません。医療記録や報告書など、精度と速度が要求される用途に適する。
Superwhisper Mac オフライン 有料（サブスクまたは買い切り、15分試用あり） Whisper.cppを利用したmacOSアプリで、日本語認識や日本語→英語翻訳が可能。1～2行程度の短文の認識が得意で、MacBook Air M2では2～3秒で変換が完了する。Macの標準音声入力とは異なり他のアプリを開いたまま音声入力でき、結果をクリップボードにコピーして自動貼付も可能。完全オフラインなので患者情報の取り扱いにも安心。
Voice In (Chrome拡張) Windows/Mac/ChromeOS オンライン 基本無料（Voice In Plusは有料） Google Chrome / Microsoft Edge で動作する拡張機能で、インストール後に言語として日本語を選択し、マイクアイコンを押して発話するとウェブ上のテキスト欄に音声をリアルタイム入力できる。句読点や特殊文字の音声コマンドが用意されており、Plus版ではカスタムコマンドの登録や長時間入力が可能。多数のウェブサービスで音声入力を使いたい人に便利。
Speechnotes Windows(ブラウザ)/Android/iOS オンライン（オフラインモードあり） 基本無料（広告あり）、プレミアムあり 無料で無制限の連続入力ができ、内蔵キーボードで句読点や記号を補助入力できる。音声コマンドで改行や句読点を挿入でき、自動大文字化やスペース挿入にも対応する。Googleの音声エンジンを利用し多言語に対応。操作が簡単で医学メモの作成などに向くが、音声がGoogleに送信されるため機密性には注意が必要。

1.4 ツール選択のポイント

精度と専門用語への対応
市販の医療用語辞書を持つツールは少ないため、カスタム辞書機能があるAqua VoiceやVoice In Plusが有利です。Superwhisperは Whisper を利用しており専門用語にも強いですが短文向けです。
オフライン対応
患者情報など機密データを扱う場合、インターネット接続のないSuperwhisperは安心です。
コスト
標準機能は無料で手軽ですが精度に限界があります。Aqua Voiceはサブスク制、Superwhisperは買い切りも選択でき、頻繁に使う場合に向きます。
用途
診療記録や学会演題など長文を入力するなら遅延が少なく自動フォーマット機能や自然言語コマンドに対応したAqua Voiceがおすすめです。メールやチャットの簡易入力にはVoice InやSpeechnotesで十分です。

2. Macで使える音声入力

macOSの標準機能とサードパーティ製ツールの特徴は上表の通りですが、医師にとっては以下の点も重要です。

📋 操作性
標準の音声入力はテキストフィールドを選択した状態でないと動作せず、他のアプリ操作ができません。一方、SuperwhisperやAqua Voiceは背景で動作し、他のアプリを操作しながら文字起こしができるため、カルテやブラウザを見ながら記録を取れます。
📋 エンジンの違い
Superwhisperはオープンソース「Whisper」をMac向けに最適化し、オフラインでも高精度に日本語を認識・翻訳できます。Aqua Voiceはクラウドとクライアント側のコンテキストエンジンを組み合わせ、他社より少ない誤認識率を実現し、自然言語コマンドに対応するため文章の修正やフォーマットを音声だけで行えます。
📋 プライバシーとセキュリティ
医療情報の入力では音声データの扱いが重要です。Superwhisperは完全オフライン、Aqua Voiceは暗号化されたクラウド処理を行うと説明しており、標準機能よりもデータ管理が明示されています。

3. スマートフォンで使える音声入力

3.1 iPhone（iPad を含む）の音声入力

iOS/iPadOSのキーボードにはマイクボタンがあり、設定→一般→キーボード→音声入力をオンにすると、どのアプリでもテキスト入力欄でマイクボタンを押して話すだけで文字が入力できます。

多くの言語はデバイス上で処理されるためインターネットなしでも使えます（検索欄など一部は検索プロバイダに送信される）。句読点や絵文字を音声で指定したり、「改行」「新しい段落」と言って段落を変えることもできます。操作を終えるにはマイクボタンを再度押すか「音声入力を停止」と話すか、30秒間黙ると自動的に終了します。

医療メモ作成など外出先での利用に便利ですが、長時間の会話をリアルタイムで文字起こしするには対応アプリ（Nottaなど）の方が適しています。

3.2 Android (Gboard) の音声入力

多くのAndroid端末にはGoogle製キーボード「Gboard」が標準搭載されています。Gboardを利用した音声入力では、キーボード左端のマイクアイコンをタップし、画面に「話してください」と表示されたら日本語で話すだけでテキスト化されます。

句読点や新しい行は「とうてん」「くてん」「新しい行」などと発話することで入力でき、心を示す絵文字やハートを挿入するなどのコマンドもあります。Googleアシスタント音声入力を併用すると「最後のテキストを削除」や「メッセージを送信」などの編集コマンドや送信操作も音声で行えます。

📋 高度な音声入力機能
Pixel 6以降にはオンデバイスで処理する「高度な音声入力」機能があり、音声をローカル処理し自動句読点を挿入、タイピングと同時に音声入力ができます。対応言語には日本語も含まれ、多言語環境では話した言語を自動判別します。

3.3 スマートフォン向けのサードパーティ製アプリ

スマートフォン向けには Speechnotes などの音声入力アプリがあり、長時間の会議録音をテキスト化したり、翻訳を行ったりできます。SpeechnotesはAndroidアプリとして提供され、連続入力やオフラインモードに対応し、音声コマンドで改行・句読点を挿入できます。iOS版はブラウザベースで利用できます。

⚠️ 注意
Aqua Voiceは現在Mac/Windows向けのサービスのみで、専用のスマートフォンアプリは提供されていません（モバイルブラウザ経由での利用は可能）。そのためスマートフォンでは標準機能やSpeechnotesなどを利用し、必要であればパソコン版のAqua Voiceに連携する使い方が現実的です。

4. まとめとおすすめ

音声入力ツールは用途や環境によって選ぶべきポイントが異なります。

📋 医療現場での長文入力には
低遅延で自然言語コマンドに対応し誤認識が少ない Aqua Voice が最適です。
📋 短文のメモや翻訳には
Superwhisper が手軽で、オフライン環境でも安心して使えます。
📋 ウェブサービスへの書き込みには
Voice In や Speechnotes が便利で、無料版でも十分利用できます。
⚠️ 標準機能の限界
標準機能は無料で導入も簡単ですが、専門用語の正確な認識や長文処理では限界があります。

用途に応じたツールを併用し、効率的な記録・報告を目指しましょう。
