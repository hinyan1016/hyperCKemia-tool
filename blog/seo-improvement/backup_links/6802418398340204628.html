はじめに：医療現場におけるコミュニケーションの壁

病院の診察室。医師から告げられる病名や検査結果、治療方針。その言葉の多くが、専門用語で語られることがあります。「高血圧症ですね、降圧剤を開始しましょう」「MRIで脳梗塞の所見が認められます」「対症療法が中心になります」…。これらの言葉を聞いて、皆さんはすぐに意味を理解できるでしょうか？

医療の専門用語は、医療従事者同士が正確かつ効率的に情報を共有するためには不可欠なものです。しかし、患者さんにとっては、難解で、時に不安を煽る「壁」となってしまうことがあります。言葉の意味が分からないままでは、ご自身の病状や治療について正しく理解できず、不安ばかりが募ってしまいます。また、医師との間に認識のずれが生じ、治療への協力が得られにくくなる可能性もあります。

この記事では、特に内科と脳神経内科の領域でよく使われる専門用語を取り上げ、それらを患者さんにも理解しやすい「やさしい日本語」に言い換える方法を【増補版】として詳しくご紹介します。医療従事者の方々にとっては、日々の説明のヒントとして。患者さんやご家族にとっては、医師の説明を理解するための一助として。この記事が、医療現場におけるより良いコミュニケーションの架け橋となることを願っています。

なぜ専門用語を言い換える必要があるのか？

専門用語を分かりやすく言い換えることには、多くのメリットがあります。

患者さんの理解度の向上:

最も重要な目的です。病名、症状、検査、治療法について、患者さん自身の言葉で理解できるようになることで、漠然とした不安が具体的な理解へと変わります。

不安の軽減と信頼関係の構築:

分からない言葉は不安を増大させます。分かりやすい説明は、患者さんの不安を和らげ、「この先生はちゃんと説明してくれる」という安心感と信頼関係につながります。

治療への積極的な参加促進 (アドヒアランス向上):

なぜこの治療が必要なのか、どんな効果が期待でき、どのような注意点があるのかを理解することで、患者さんは治療に対して前向きになり、積極的に協力してくれるようになります（アドヒアランスの向上）。薬の飲み忘れを防いだり、生活習慣の改善に取り組んだりする意欲にもつながります。

インフォームド・コンセントの質の向上:

インフォームド・コンセント（説明と同意）は、医療の基本です。患者さんが十分な情報を理解した上で、自らの意思で治療方針に同意するためには、分かりやすい言葉での説明が不可欠です。専門用語の羅列では、真のインフォームド・コンセントは成立しません。

内科領域の専門用語：やさしい日本語への言い換え例【増補版】

日常の診療で出会うことの多い内科領域の用語を、より詳しく見ていきましょう。病気のこと、体のサイン（症状）、そして検査や治療について、様々な言い換え方を考えてみます。

【疾患名】

高血圧症:

  • → 「血圧が高い状態ですね。」
  • → 「血管にいつも強い圧力がかかっている状態です。」
  • → 「心臓が血液を送り出す力が強すぎるか、血管が硬くなっているのかもしれません。」

補足：「上の血圧（収縮期）が140、下の血圧（拡張期）が90を超えると高血圧と診断されることが多いです。自覚症状はなくても、放っておくと血管が傷み、将来的に心臓病や脳卒中、腎臓病のリスクが高まります。お薬や生活習慣（塩分を控える、運動するなど）の見直しで、適切な血圧を保つことが大切です。」

糖尿病:

  • → 「血糖値（血液中の糖分の濃度）が高い状態が続いている病気です。」
  • → 「食べ物から摂った糖分を、エネルギーとしてうまく利用できなくなっています。」
  • → 「インスリンという、血糖値を下げるホルモンの働きが悪くなったり、量が足りなくなったりしています。」

補足：「のどが渇く、おしっこの回数が増える、疲れやすいといった症状が出ることがありますが、初期は無症状のことも多いです。血糖値が高い状態が続くと、全身の細い血管や神経が傷つき、目（網膜症）、腎臓（腎症）、神経（神経障害）などに合併症を引き起こす可能性があります。食事療法、運動療法、そして必要に応じてお薬（飲み薬やインスリン注射）で血糖値をコントロールします。」

高脂血症 (脂質異常症):

  • → 「血液の中の『あぶら』の成分（コレステロールや中性脂肪）のバランスが崩れている状態です。」
  • → 「悪玉コレステロール（LDL）や中性脂肪が多すぎる、または善玉コレステロール（HDL）が少なすぎる状態を指します。」
  • → 「血液がドロドロになりやすく、血管の壁に『プラーク』という塊ができやすい状態です。」

補足：「これも自覚症状はほとんどありませんが、動脈硬化（血管が硬く、狭くなること）を進める大きな原因となり、心筋梗塞や脳梗塞につながります。食事（脂っこいものや甘いものを控える）、運動、禁煙などが重要で、それでも改善しない場合はお薬を使います。」

心不全:

  • → 「心臓のポンプ機能（血液を全身に送り出す力）が弱ってしまい、体が必要とする量の血液を十分に送り出せなくなった状態です。」
  • → 「心臓が疲れてしまっている、とイメージしてください。」

補足：「原因は様々（心筋梗塞の後、高血圧、心臓弁膜症など）ですが、結果として体に水が溜まりやすくなり、息切れ、むくみ、疲れやすさなどの症状が出ます。心臓の負担を減らすお薬や、原因となる病気の治療、塩分・水分制限などを行います。完全に治るというより、症状をコントロールし、うまく付き合っていくことが目標になります。」

狭心症:

  • → 「心臓に栄養や酸素を送る血管（冠動脈）が動脈硬化などで狭くなり、心臓の筋肉への血流が一時的に不足する状態です。」
  • → 「階段を上ったり、重いものを持ったりした時に、一時的に胸が締め付けられるような痛みや圧迫感が出ることが多いです。」

補足：「痛みは通常、数分〜15分程度で治まります。心筋梗塞（血管が完全に詰まる）の前触れである可能性もあるため、検査が必要です。お薬で発作を予防したり、カテーテル治療（血管を広げる）を行ったりします。」

心筋梗塞:

  • → 「心臓の血管（冠動脈）が完全に詰まってしまい、心臓の筋肉の一部に血液が届かなくなり、その部分の筋肉が壊死（えし）してしまう病気です。」
  • → 「突然の激しい胸の痛み（締め付けられる、焼けつくような）が長く続くのが特徴です。」（※冷や汗、吐き気、呼吸困難を伴うことも）

補足：「命に関わる緊急性の高い病気です。一刻も早く詰まった血管の血流を再開させる治療（カテーテル治療や血栓溶解療法）が必要です。」

不整脈:

  • → 「心臓のリズム（脈拍）が、規則正しくなくなったり、速すぎたり（頻脈）、遅すぎたり（徐脈）する状態の総称です。」
  • → 「心臓の電気信号の伝わり方に異常が起きています。」

補足：「動悸（ドキドキする）、脈が飛ぶ感じ、めまい、ふらつき、失神、胸の不快感など、症状は様々です。心配のないタイプもありますが、中には脳梗塞の原因になったり（心房細動など）、命に関わる危険なタイプもあるため、心電図などで種類を特定することが重要です。」

肺炎:

  • → 「肺の中に細菌やウイルスなどが入り込んで、炎症を起こしている状態です。」
  • → 「肺が腫れて、そこに水や膿が溜まることもあります。」

補足：「症状は、咳、痰（黄色や緑色）、発熱、胸の痛み、息苦しさなどです。高齢者や持病のある方は重症化しやすいので注意が必要です。原因菌に合わせた抗生物質や、症状を和らげる薬で治療します。誤嚥（食べ物や唾液が気管に入ること）が原因で起こる肺炎（誤嚥性肺炎）も増えています。」

慢性閉塞性肺疾患 (COPD):

  • → 「主に長年の喫煙が原因で、肺の組織が壊れたり（肺気腫）、気管支に炎症が起きたりして、空気の通り道が狭くなり、息を吐き出しにくくなる病気です。」
  • → 「『タバコ病』とも呼ばれます。」

補足：「主な症状は、慢性の咳や痰、体を動かした時の息切れです。進行すると、少し動いただけでも息苦しくなります。禁煙が最も重要で、気管支を広げる吸入薬などで症状を和らげます。」

気管支喘息:

  • → 「空気の通り道（気管支）が、アレルギーや刺激に対して過敏になっていて、発作的に狭くなり、呼吸が苦しくなる状態を繰り返す病気です。」
  • → 「気管支の粘膜が慢性的に炎症を起こしています。」

補足：「発作時には、咳、痰、ゼーゼー・ヒューヒューという呼吸音（喘鳴）、息苦しさが出ます。普段から気管支の炎症を抑える吸入ステロイド薬などを使い、発作を予防することが治療の基本です。」

【症状】

  • 胸痛: → 胸の痛み、締め付けられる感じ、圧迫感、焼けつく感じ (いつから？どんな時に？どのくらいの時間？なども確認)
  • 呼吸困難/息切れ: → 息苦しさ、息が吸いにくい/吐きにくい、呼吸が浅い、ハアハアする (じっとしていても苦しいか？動くと苦しいか？横になると苦しいか？)
  • 動悸: → ドキドキする、心臓がバクバクする、脈が飛ぶ、脈が乱れる感じ
  • 浮腫: → むくみ、腫れぼったい感じ (どこが？いつから？指で押すと跡が残るか？)
  • 倦怠感: → だるさ、体が重い、疲れやすい、気力がない
  • 発熱: → 熱がある、体温が高い、寒気がする
  • 悪心・嘔吐: → 吐き気、むかつき、胃が気持ち悪い、もどす
  • 食欲不振: → 食欲がない、食べたくない、すぐに満腹になる
  • 腹痛: → お腹の痛み (どのあたりが？いつから？どんな痛み(キリキリ、シクシク、ズキズキ)？食事との関係は？)
  • 下痢: → お腹がゆるい、水のような便、頻繁に便意がある
  • 便秘: → 便が出にくい、便が硬い、排便の回数が少ない、残便感がある
  • 吐血: → 血を吐くこと (鮮やかな赤か、黒っぽいか？)
  • 下血/黒色便(タール便): → お尻から血が出ること (鮮やかな赤か、便に混じっているか？)、黒くてドロっとした便が出ること
  • 黄疸: → 皮膚や白目が黄色くなること
  • 腹水: → お腹に水がたまって張ること、体重が急に増えること
  • 嗄声 (させい): → 声がかれること、声が出しにくいこと
  • 咳嗽 (がいそう): → 咳が出ること (痰は絡むか？乾いた咳か？いつから？)
  • 喀痰 (かくたん): → 痰が出ること (色は？量は？粘り気は？)

【検査・治療】

  • 血液検査: → 「血液を少し採って、赤血球や白血球の数、肝臓や腎臓の機能、コレステロールや血糖値、炎症の程度などを調べる基本的な検査です。」
  • 尿検査: → 「尿を採って、糖やタンパク質が出ていないか、血が混じっていないか、感染がないかなどを調べます。腎臓や膀胱の病気の手がかりになります。」
  • 便検査 (便潜血検査など): → 「便を少量採って、目に見えない微量の出血がないか（大腸がん検診など）、細菌感染がないかなどを調べます。」
  • 心電図 (ECG/EKG): → 「胸や手足に電極を付けて、心臓が動く時の微弱な電気信号を記録し、波形グラフで見る検査です。不整脈や狭心症、心筋梗塞などの診断に役立ちます。」
  • 胸部X線 (レントゲン): → 「胸にX線を当てて、肺（肺炎、肺がん、気胸など）、心臓（心拡大など）、骨（肋骨骨折など）の状態を見る検査です。」
  • 腹部X線 (レントゲン): → 「お腹にX線を当てて、腸内のガスの状態（腸閉塞など）や、異常な石灰化などを見る検査です。」
  • CT検査 (Computed Tomography): → 「X線を使って体を輪切りにしたような詳細な断面画像を見る検査です。レントゲンでは分からない小さな病変や、臓器（脳、肺、肝臓、膵臓、腎臓など）の状態を詳しく調べることができます。造影剤という薬を注射して、血管や病変をよりはっきり見せることもあります。」
  • MRI検査 (Magnetic Resonance Imaging): → 「強力な磁石と電波を使って、体の様々な断面を詳しく見る検査です。CTとは異なる情報が得られ、特に脳、脊髄、関節、筋肉、骨盤内の臓器などの病変を描出するのに優れています。放射線被ばくはありません。造影剤を使うこともあります。」（※ペースメーカーや体内に金属がある方は受けられない場合があることを確認）

脳神経内科領域の専門用語：やさしい日本語への言い換え例【増補版】

脳や神経に関わる病気は、目に見えない部分だけに、より分かりやすい説明が求められます。

【疾患名】

脳梗塞:

  • → 「脳の血管が血の塊（血栓）などで詰まってしまい、その先の脳細胞に血液（酸素や栄養）が届かなくなり、脳の一部がダメージを受けてしまう病気です。」
  • → 「詰まる血管の場所や太さによって、様々な症状（麻痺、しびれ、言葉の障害、めまいなど）が突然現れます。」

補足：「原因として、心臓でできた血栓が飛んでくる（心原性脳塞栓症）、脳の血管の動脈硬化が原因（アテローム血栓性脳梗塞）、細い血管が詰まる（ラクナ梗塞）などがあります。発症したら、できるだけ早く治療を開始することが、後遺症を軽くするために重要です。再発予防のために、血液をサラサラにする薬（抗血小板薬や抗凝固薬）や、生活習慣病（高血圧、糖尿病、脂質異常症）の管理が欠かせません。」

脳出血:

  • → 「脳の中の細い血管が破れて、脳内に出血し、血の塊（血腫）ができる病気です。」
  • → 「高血圧が長年続くことで血管がもろくなり、破れやすくなるのが主な原因です。」

補足：「出血した場所や量によって、症状（突然の頭痛、麻痺、意識障害、嘔吐など）は大きく異なります。血圧のコントロールが非常に重要です。出血が大きい場合は、手術で血腫を取り除くこともあります。」

【症状】

  • 麻痺 (片麻痺、四肢麻痺、顔面神経麻痺など): → 手足や顔などが動かせなくなる、力が入りにくくなる、動きがぎこちない (例：「右半身がだらんと力が抜けた」「顔の半分がゆがんで、うまく閉じられない」)
  • しびれ (痺れ): → ジンジン、ピリピリ、チクチク、電気が走るような感じ、感覚が鈍い、触っても感じにくい、正座の後のような感じ (どこが？いつから？)
  • 構音障害: → ろれつが回らない、言葉がもつれる、滑舌が悪い、発音が不明瞭になる
  • 失語症 (運動性失語/ブローカ失語、感覚性失語/ウェルニッケ失語など):
    • → (全般) 言葉を理解したり、話したり、読んだり、書いたりする能力が障害されること。
    • → (運動性) 言いたいことは分かっているのに、言葉がうまく出てこない、たどたどしい話し方になる。
    • → (感覚性) 言葉は流暢に話せるが、内容がちぐはぐになったり、相手の言うことの意味が理解できなかったりする。

【検査・治療】

  • 頭部CT検査: → 「X線を使って、脳の断面を撮影する検査です。脳出血、くも膜下出血、大きな脳梗塞、脳腫瘍などを素早く見つけることができます。検査時間は短いです。」
  • 頭部MRI検査: → 「強力な磁石と電波を使って、脳を様々な角度から詳しく見る検査です。CTよりも細かい病変や、発症して間もない脳梗塞、脳腫瘍、炎症（脳炎、多発性硬化症など）、脳の萎縮の程度などを評価するのに優れています。検査時間がCTより長く（30分程度）、大きな音がします。」（※閉所恐怖症の方への配慮、金属の確認）

言い換えの際の注意点：より良いコミュニケーションのために

専門用語を分かりやすく言い換える際には、いくつか心に留めておきたい点があります。

正確性を損なわない:

分かりやすさを追求するあまり、医学的な意味合いが大きく歪んでしまっては本末転倒です。基本的な事実は正確に伝え、誤解を招かないように注意しましょう。

患者さんの理解度を確認しながら:

一方的に説明するのではなく、「ここまでの説明で、何か分かりにくいところはありますか？」「今の言葉、どういう意味か伝わりましたか？」など、対話を心がけ、患者さんの表情や反応を見ながら、理解度を確認することが大切です。

図や模型、資料の活用:

言葉だけではイメージしにくいことも、簡単な図を描いたり、体の模型を使ったり、説明用のパンフレットを見せたりすることで、格段に理解しやすくなります。「百聞は一見にしかず」です。

適切な比喩表現:

「血管が詰まる」を「水道管が詰まるようなイメージ」、「神経の伝達」を「電気のコード」に例えるなど、身近なものに例えると理解の助けになることがあります。ただし、例えが不適切だと逆に混乱させる可能性もあるため、慎重に選びましょう。

ネガティブな言葉遣いを避ける:

病気の説明は、どうしても悪い知らせを伴うことがあります。しかし、「もう治りません」「どんどん悪くなるだけです」といった希望を失わせるような断定的な表現や、不安を過度に煽る言葉は避け、「今の医学では完全に治すことは難しいですが、症状を和らげる治療法があります」「進行を遅らせるために、一緒に頑張りましょう」など、前向きな姿勢で、寄り添う気持ちを込めて伝えることが大切です。

メモや要点を書いた紙を渡す:

一度の説明ですべてを記憶するのは困難です。特に高齢の方や、ショックを受けている方には、病名や治療の要点、次回の受診日などをメモに書いて渡すと、後で落ち着いて確認でき、安心感につながります。

一方的な「言い換え」ではなく「対話」:

大切なのは、単に難しい言葉を簡単な言葉に置き換える作業ではありません。患者さんが何を知りたいのか、何を不安に思っているのかを汲み取り、その疑問や不安に寄り添いながら、対話を通じて理解を深めていくプロセスそのものです。

まとめ：分かりやすい言葉は、信頼の架け橋

医療現場における専門用語は、時に患者さんとの間に見えない壁を作ってしまいます。その壁を取り払い、患者さんが安心して医療を受けられるようにするためには、私たち医療従事者が、専門用語を「やさしい日本語」に翻訳し、対話を重ねていく努力が不可欠です。

今回ご紹介した言い換え例は、あくまで一例です。大切なのは、目の前の患者さんの年齢、理解力、そして何を知りたいと思っているかに合わせて、言葉を選び、丁寧に説明を尽くす姿勢です。

そして、患者さんやご家族の皆様へ。医師の説明で分からないこと、疑問に思うことがあれば、決して遠慮せず、「今の言葉、もう少し分かりやすく教えてください」「それは、どういう意味ですか？」と質問してください。質問することは、ご自身の治療に主体的に関わるための第一歩です。

分かりやすい言葉によるコミュニケーションは、医師と患者さんの間の信頼関係を深め、より良い医療を実現するための重要な鍵となります。この記事が、その一助となれば幸いです。
