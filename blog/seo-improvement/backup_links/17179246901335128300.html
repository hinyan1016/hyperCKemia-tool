WAISとは：成人の知能を多面的に評価する検査

WAIS（Wechsler Adult Intelligence Scale：ウェクスラー式成人知能検査）は、16歳以上の知的機能を、複数の課題（下位検査）で評価し、全体のIQ（FSIQ）と領域別の得点（指標得点など）からプロフィールを読み取る検査です[1]。

日本で現在「最新の日本版」として提供されているのはWAIS-IVで、16歳0か月〜90歳11か月を対象とし、4つの領域（VCI/PRI/WMI/PSI）とFSIQを算出します[1]。

3つの版の「いちばん大きい違い」

WAIS-R

基本は「言語性IQ（VIQ）」「動作性IQ（PIQ）」「全検査IQ」の3つを算出する構造です。

WAIS-III

VIQ/PIQを残しつつ、4つの群指数（言語理解・知覚統合・作動記憶・処理速度）を導入しました[2]。

WAIS-IV

VIQ/PIQを廃止し、FSIQ＋4指標（VCI/PRI/WMI/PSI）を中心に解釈する構造へ大きく転換しました[2]。

各版で何が具体的に変わったか

WAIS-R → WAIS-III：プロフィールが読みやすくなった（4群指数の導入）

WAIS-IIIでは、従来のVIQ/PIQに加えて、以下の4つの群指数が得られるようになり、得意・不得意の理由を"領域"として説明しやすくなりました[6]。

📋 WAIS-IIIの4群指数 ・言語理解（VC）
・知覚統合（PO）
・作動記憶（WM）
・処理速度（PS）

また、WAIS-IIIは14の下位検査から構成されることが明記されています[6]。

WAIS-III → WAIS-IV：VIQ/PIQをやめ、4指標中心の設計へ（より理論整合的）

WAIS-IVの大きな転換点は、言語性IQ（VIQ）と動作性IQ（PIQ）が廃止され、IQは基本的にFSIQ中心になった点です[2]。

そのうえで、領域は以下の4指標で整理されます。

📋 WAIS-IVの4指標 ・VCI（言語理解）
・PRI（知覚推理）
・WMI（ワーキングメモリー）
・PSI（処理速度）

WAIS-IVは、通常、基本10検査でFSIQ＋4指標を算出し、補助得点としてGAI（一般知的能力指標）も扱えます[1]。

下位検査の入れ替え詳細（クリックして展開）

日本文化科学社の製品説明では、WAIS-IVで以下の変更が明記されています[1]。

新しい下位検査の追加：

・「パズル」：視覚的推理能力を測定
・「バランス」：量的推理能力を測定
・「絵の抹消」：処理速度を測定

既存検査の改良：

・「数唱」に「数唱：数整列」が追加

削除された下位検査：

・WAIS-IIIにあった「絵画配列」「組合せ」などが削除

年齢上限と実施負担の改善

WAIS-IVは適用上限が90歳11か月まで拡大し、さらに実施時間が短縮されるような改良（中止条件の短縮など）が盛り込まれています[1]。

日本での実施状況

日本版が正式に整備され、医療でも「検査」として運用されている

日本文化科学社の説明では、WAIS-IVは医療分野でも用いられ、医科診療報酬点数が450点（D283-3）と明記されています[1]。また、使用者レベルC（高度な専門性を要する区分）として示されています[1]。

日本版の刊行の流れ（版の更新）

現場の「移行」の実態

流通・供給面では、WAIS-IIIはコンプリートセット等が販売終了（2023年3月末）で、記録用紙セットは期限付きで残る、という情報が出ています[7]。このため、新規の実施はWAIS-IVが中心になっていく（あるいは既になっている）と考えるのが自然です。

⚠️ 注意 ただし臨床では、施設が既に持っている用具・運用や、研究・過去データとの連続性などの理由で、一定期間は複数版が混在し得ます。新規導入はWAIS-IVが基本です。

臨床での注意点

🚨 重要な臨床上の注意 ・版が違うと、同じ「IQ」でも前提（構成・ノルム）が違うので、単純比較は危険です（特に経時比較）
・WAIS-IVではVIQ/PIQが出ないため、「言語＞動作」のような古い言い方は、VCI/PRI/WMI/PSIのどこに差があるかへ読み替えるのが実務的です[2]

高次脳機能障害や認知症評価では、WAIS単独で結論を出すより、病歴・神経所見・他検査（記憶、注意、遂行、言語など）と統合して判断するのが安全です（これは版に関係なく共通）。

WAIS-R/III/IV と WAB の違い

WAISと混同されやすい検査にWAB（Western Aphasia Battery）があります。これは失語症の評価専用検査であり、WAISとは目的・対象・評価内容が全く異なります。

WABが評価する内容

・失語症があるか？
・どのタイプか？（Broca失語、Wernicke失語など）
・どの程度重いか？

WABの主な評価項目

・自発話
・聴理解
・復唱
・呼称
・読み・書き

📋 WAISとWABの決定的な違い WAIS：IQを測る → 認知全般を評価
WAB：IQは測らない → 言語機能に特化
観点 WAIS-R / III / IV WAB
目的 知能評価 失語症評価
主な対象 認知全般 言語機能
脳疾患との関係 高次脳機能障害全般 左半球障害・脳卒中後など
IQ算出 あり なし
臨床使用場面 認知症、発達、外傷 脳卒中後の失語症
臨床での使い分け（専門医目線）

「この人、頭が全体的に落ちている？」
→ WAISを実施

「言葉だけおかしい？」
→ WABを実施

脳卒中後で両方疑わしい
→ WAB＋WAIS（またはMMSEなど）併用

まとめ

📋 臨床で押さえておくべきポイント ・WAIS-R／III＝知能検査（旧版）
・WAIS-IV＝知能検査（現行の最新版、VIQ/PIQを廃止し4指標中心）
・WAB＝失語症評価（WAISとは別物）
・WAIS-IVの方が神経内科的に情報量が多い（脳局在との対応づけがしやすい）
・日本ではWAIS-IVが現行版として医療現場で運用されている（診療報酬450点）
