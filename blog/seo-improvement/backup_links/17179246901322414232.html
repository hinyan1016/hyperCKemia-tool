NST（Nutrition Support Team：栄養サポートチーム）回診でベッドサイドに行った際、患者に何を聞き、何を観察・診察すればよいかを「明日すぐに使えるレベル」でまとめました。本ガイドは実践的な問診項目と診察ポイントを体系的に整理し、効率的かつ包括的な栄養評価を可能にします。

NST回診で評価すべき6つの要素

NSTが評価すべき内容は以下の6つの要素が中心となります：

📋 NST評価の6つの柱
  • ①摂取状況：経口摂取量・食欲・食事時間
  • ②嚥下機能：むせ・嚥下困難の有無
  • ③消化器症状：嘔気・腹部膨満・排便状況
  • ④身体所見：浮腫・褥瘡・脱水徴候
  • ⑤サルコペニア指標：筋肉量・筋力低下
  • ⑥チューブ類や点滴管理：経腸栄養・輸液ラインの状態

これらを体系的に評価することで、栄養状態の包括的な把握と適切な介入が可能になります。

ベッドサイドで患者に聞くこと（問診項目）

以下の項目は、実際に口頭で患者に確認すべき内容です。患者の主観的な訴えから栄養摂取の障害因子を見抜くことが重要です。

1. 食事・経口摂取について

質問項目 評価目的
今どのくらい食べられていますか？（量・時間） 経口摂取量の実測・見積もり
栄養投与量の妥当性評価
飲み込みにくさやむせはありますか？ 嚥下障害の確認
誤嚥リスクの早期発見
食事中に疲れやすいですか？ 咀嚼力低下・サルコペニアのサイン
食事持久力の評価
食事の味や匂いは感じますか？ 嗅覚障害・味覚障害の確認
食欲低下の原因探索
食欲はありますか？ 摂取不良の原因探索
精神的要因・病態の影響評価
食事の形態（軟菜・ミキサー・とろみ）は合っていますか？ 食事形態の適正評価
嚥下機能に応じた調整の必要性

2. 水分摂取・口腔状況

以下の項目を確認します：

  • 水分はどれくらい飲めていますか？ → 脱水リスクの評価
  • のどの渇き、口の乾燥はありますか？ → 脱水の自覚症状確認
  • 口の痛み・口内炎の有無 → 経口摂取阻害因子の把握
⚠️ 注意
高齢者では口渇感が鈍麻していることが多く、主観的な訴えだけでなく客観的な脱水徴候（後述）の確認が必須です。

3. 消化器症状

質問項目 評価の意図
吐き気や嘔吐はありますか？ 栄養投与の耐性評価
胃内容の停滞・逆流リスク
お腹の張りはありますか？ 腸蠕動低下・イレウスのサイン
経腸栄養の継続可否判断
排便はいつありましたか？ 便秘の確認
摂取低下・腹部膨満の原因
下痢はありますか？ 経腸栄養過多・浸透圧性の影響
投与速度・濃度の調整必要性

4. 倦怠感・筋力低下（サルコペニア評価）

サルコペニアの早期発見のため、以下の質問が有用です：

  • 最近、立ち上がりがしんどくなりましたか？ → 下肢筋力低下の自覚
  • 歩いていて疲れやすくなりましたか？ → 持久力低下・全身性筋力低下
📋 ポイント
患者の主観的な筋力低下の訴えと、後述する客観的な筋肉量評価を組み合わせることで、サルコペニアの包括的な評価が可能になります。

ベッドサイドで診察・観察すること

以下は専門医目線での診察・観察チェックリストです。視診・触診・聴診を組み合わせた系統的な評価を行います。

1. 全身状態・栄養状態

栄養状態の包括的評価のため、以下を観察します：

  • 体重・むくみ（浮腫）
    • 体重変化率の確認（1週間で1-2%、1ヶ月で5%以上の減少は有意）
    • 下肢・仙骨部の圧痕性浮腫の有無
  • 皮膚の乾燥・褥瘡・皮膚トラブル
    • 低栄養による創傷治癒遅延の評価
    • 褥瘡のステージング（NPUAP分類）
  • 筋肉量の評価
    • 上腕三頭筋、手背の筋萎縮：視覚的な筋量評価
    • 大腿部、ふくらはぎの太さ（下腿周囲径）：下腿周囲径30cm未満はサルコペニアのカットオフ値
  • 結膜の蒼白（貧血）
    • 眼瞼結膜の色調確認
    • 低栄養性貧血・鉄欠乏性貧血の示唆
  • 脱水サイン
    • 舌の乾燥：舌表面の湿潤度確認
    • 皮膚ツルゴール（皮膚弾力）：前腕内側の皮膚を軽くつまんで戻りを確認
    • 眼球陥凹、口腔粘膜の乾燥
📋 下腿周囲径の測定方法
患者を仰臥位にし、膝窩の最も太い部分で測定。男性34cm未満、女性33cm未満でサルコペニアのリスクが高いとされます（日本人の基準値）。

2. 嚥下機能の観察（簡易ベッドサイド評価）

嚥下機能の評価は、誤嚥性肺炎予防の要です。以下の観察ポイントを確認します：

  • むせ込み・咳反射の有無
    • 飲水時の観察（3ml水飲みテストなど）
    • 食事中のむせの頻度
  • 声の湿り（ウェットボイス）
    • 発声時にゴロゴロした湿性の声 → 咽頭残留・誤嚥のサイン
  • 頸部の可動域・姿勢
    • 頸部伸展位での嚥下は誤嚥リスク増大
    • 適切な頸部前屈（chin-down）姿勢の確認
  • 食事中の疲労感
    • 途中でペースが落ちる → 嚥下筋力低下・咀嚼疲労
  • 口腔内の残渣
    • 食後の口腔内に食物残渣 → 口腔期嚥下障害
    • 口腔ケア不足による誤嚥リスク増大
⚠️ 重要
わずかな変化が誤嚥リスクを示すため、言語聴覚士（ST）介入の判断に重要です。不顕性誤嚥（silent aspiration）ではむせが出ないこともあり、声質の変化や食事時間の延長などの微細なサインを見逃さないことが重要です。

3. 口腔内の観察

口腔内の状態は嚥下機能と密接に関連します：

  • 舌苔の付着 → 口腔ケア不足、脱水の指標
  • 歯の状態（義歯の適合） → 咀嚼機能への影響
  • 口腔乾燥 → 薬剤性・脱水・シェーグレン症候群など
  • 残渣 → 嚥下機能低下、口腔ケア不足
📋 NSTがよく忘れがちだが最重要
口腔内の実見は、誤嚥性肺炎対策として極めて重要です。口腔内細菌の増殖が誤嚥性肺炎の直接的な原因となるため、口腔ケアの徹底が必須です。

4. 消化器系の状態

腹部の視診・聴診・触診により消化管機能を評価します：

観察項目 評価内容
腹部膨満の有無 視診・触診での腹部膨隆
腸管ガス貯留・腹水の確認
腸蠕動音 消化吸収機能の目安
イレウス・腸管麻痺の評価
経腸チューブの管理 ・位置・固定状況の確認
・流量・残量（残椀）の測定
・チューブの屈曲・閉塞の有無
胃ろう部の皮膚トラブル 瘻孔周囲の発赤・浸出液・肉芽形成
バンパー埋没症候群の確認
⚠️ 残椀（胃内残留）の評価
経腸栄養投与前に胃内残留量を確認することが重要です。200ml以上の残留は胃内容停滞を示唆し、投与速度の減速や投与間隔の延長を検討します。

5. 点滴・輸液ライン

末梢静脈栄養（PPN）や中心静脈栄養（TPN）の管理状況を確認：

  • 血管外漏出
    • 刺入部の腫脹・発赤・疼痛
    • 高浸透圧輸液の漏出は組織壊死のリスク
  • 流速・ルートトラブル
    • 閉塞・屈曲・三方活栓の開閉確認
    • 予定投与量が確実に投与されているか
  • 輸液の内容と量
    • 必要エネルギー量との比較
    • 電解質・微量元素の補充状況

6. バイタル・呼吸

全身状態の把握と感染症の早期発見のため：

  • 呼吸数・SpO₂
    • 誤嚥性肺炎の早期サイン
    • SpO₂ 90%未満は要注意
  • 発熱の有無
    • 感染 → 代謝亢進 → 摂取低下
    • カテーテル関連血流感染（CRBSI）のリスク

NST回診時に特に重要な3つのポイント

🚨 NST回診で必ずベッドサイドでチェックすべき3つ

① 「食べられない理由が身体か、気持ちか、環境か」を見抜く

摂取不良の原因を多角的に評価します：

  • 病態由来：疼痛、消化器症状、呼吸困難など
  • 嚥下機能：嚥下障害、口腔内トラブル
  • 精神的要因：うつ・意欲低下、認知機能低下
  • 環境因子：ベッド上姿勢、食事形態、介助の有無
📋 ポイント
「食欲がない」という訴えの背景には多様な原因が隠れています。身体的・精神的・環境的要因を総合的に評価し、適切な介入ポイントを見出すことがNSTの役割です。

② 目で見た筋量と、患者の訴えのズレ

主観と客観のギャップに注目します：

  • 「食べてると言うけど実は摂取量少ない」
    • 患者の認識と実際の摂取量の乖離
    • 食事記録との照合が重要
  • 「元気そうに見えるが筋量は落ちている」
    • 会話や表情は良好でも、サルコペニアが進行していることがある
    • 筋肉量の客観的評価（上腕・下腿周囲径、握力測定など）が必須

③ 口腔内の実見

NSTがよく忘れがちだが、誤嚥性肺炎対策として最重要です：

  • 口腔内細菌の増殖が誤嚥性肺炎の直接的原因
  • 舌苔・残渣・義歯の状態確認
  • 口腔ケアの指導・介入の必要性判断
⚠️ 見落としがちな重要ポイント
栄養状態の評価に集中するあまり、口腔内の観察が疎かになりがちです。しかし、口腔ケア不足は誤嚥性肺炎の最大のリスク因子であり、必ず確認すべき項目です。

まとめ：効果的なNST回診のための実践ポイント

NST回診の成功のカギ

  • 系統的な評価：6つの評価要素（摂取、嚥下、消化器、身体所見、サルコペニア、チューブ管理）を漏れなく確認
  • 主観と客観の統合：患者の訴え（問診）と身体所見（診察）を照合し、ギャップを見抜く
  • 口腔内の重視：見落としがちだが、誤嚥性肺炎予防の最重要ポイント
  • 多職種連携：ST（嚥下評価）、歯科衛生士（口腔ケア）、看護師（日常観察）との情報共有

ベッドサイド評価で得られる情報の活用

ベッドサイドで得られた情報は、以下の栄養管理計画に直結します：

  • 経口摂取量の評価 → 必要栄養量の算定
  • 嚥下機能の評価 → 食事形態の調整、ST介入
  • 消化器症状の評価 → 経腸栄養の投与方法変更
  • サルコペニアの評価 → 蛋白質・エネルギー投与量の増量、リハビリ介入
  • 脱水・浮腫の評価 → 水分バランスの調整

これらの評価を通じて、個々の患者に最適化された栄養管理計画を立案し、定期的に再評価することが、NST回診の本質的な価値です。
